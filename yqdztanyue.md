# 2026款一汽大众探岳 动力技术参数介绍
## 一、整车基础概况
探岳基于MQB Evo平台打造，定位中型城市SUV，轴距2731mm；全系搭载**7速湿式双离合DQ381变速箱**，分三套动力：300TSI 1.5T前驱、330TSI 2.0T低功前驱、380TSI 2.0T高功+第五代4MOTION智能四驱；1.5T可加注92#汽油，2.0T建议95#；底盘全系前麦弗逊+后五连杆独立悬架，德系韧性底盘调校，兼顾高速稳定与家用舒适。

## 二、三套动力总成详细参数
### 1. 300TSI 1.5T Evo2 前驱（入门经济版）
- 发动机型号：EA211-DSV 1.5L直列四缸涡轮增压，米勒循环、VTG可变截面涡轮、350MPa高压直喷
- 最大功率：118kW（160Ps）/5500rpm
- 峰值扭矩：250N·m /1750–4000rpm，宽扭矩区间低速响应平顺
- 变速箱：DQ381 7速湿式双离合，独立油冷散热，拥堵蠕行不易过热
- 驱动形式：前置前驱
- 官方0-100km/h加速：9.4s
- 最高电子限速：210km/h
- WLTC综合油耗：6.39–6.7L/100km
- 油箱容积60L，满油综合续航约890km
- 适用场景：城市通勤、日常代步，追求低用车成本用户

### 2. 330TSI 2.0T EA888低功率 前驱（家用走量主力）
- 发动机型号：EA888-DTH 2.0L直列四缸涡轮增压，水冷涡轮、冷却EGR
- 最大功率：137kW（186Ps）/4100–6000rpm
- 峰值扭矩：320N·m /1500–4000rpm，1500转即可输出峰值扭矩，满载爬坡、高速超车底气充足
- 变速箱：DQ381强化版7速湿式双离合
- 驱动形式：前置前驱
- 官方0-100km/h加速：8.3s
- WLTC综合油耗：6.84–6.95L/100km
- 油箱容积60L，满油综合续航约860km
- 适用场景：全家出行、长途高速，兼顾动力与油耗均衡

### 3. 380TSI 2.0T EA888高功率 + 4MOTION四驱（性能顶配）
- 发动机型号：EA888-DTJ 2.0T高功率涡轮，强化冷却系统，持续高负荷动力无衰减
- 最大功率：162kW（220Ps）/4500–6200rpm
- 峰值扭矩：350N·m /1600–4300rpm，中段再加速能力突出
- 变速箱：DQ381高性能7速湿式双离合
- 驱动形式：前置第五代Haldex电控适时四驱
- 官方0-100km/h加速：7.4s
- WLTC综合油耗：7.96L/100km
- 油箱容积60L，满油综合续航约750km
- 最高电子限速：220km/h

#### 第五代4MOTION四驱核心特性
1. 电控多片离合器中央差速器，毫秒级动态分配前后扭矩，极限可100%后轮输出；
2. 标配自动/舒适/运动/经济/雪地/越野6种驾驶模式，同步调整动力、换挡、四驱介入强度；
3. 轮间EDS电子限滑，单侧车轮打滑自动制动，具备交叉轴、雪地轻度脱困能力；
4. 高速巡航默认偏前驱，降低传动损耗，兼顾操控与燃油经济性。

## 四、底盘、制动与驾驶特性
1. **悬架结构**：全系前麦弗逊独立悬架（液压衬套）+后四连杆多连杆独立悬架，底盘扭转刚度28000N·m/deg；德系偏韧性调校，高速行驶厚重不发飘，过弯侧倾抑制优秀，细碎颠簸过滤到位。
2. **通过性**：空载离地间隙205mm，满载158mm，可应对坑洼、非铺装轻度越野。
3. **制动系统**：四轮通风盘制动，标配ESC车身稳定、上坡辅助、陡坡缓降；无电驱能量回收，制动踏板线性，燃油车传统舒适制动逻辑。
4. **转向**：电动助力转向，低速轻盈、高速加重，城市窄路、停车更灵活。

## 五、动力安全防护机制
1. EA211/EA888发动机水冷循环温控，长时间满载爬坡、高温拥堵分级限制输出功率，涡轮熄火延时散热，降低热衰减；
2. DQ381湿式双离合内置独立油冷，城市蠕行、连续山路不易触发过热保护，耐久性优于干式双离合；
3. 4MOTION四驱版本联动ESC稳定系统，雨雪湿滑路面动态分配扭矩，抑制推头、甩尾、高速横风跑偏；
4. 多重电控分级保护：爆震、涡轮超压、冷却系统故障自动限制动力输出，保护发动机硬件；
5. 全系配备主动刹车、牵引力控制，提升湿滑路面行驶安全。

## 六、发动机核心技术亮点
1. **1.5T Evo2**：VTG可变截面涡轮，低转速无迟滞；米勒循环提升热效率，日常通勤油耗更低，支持92#汽油降低用车成本。
2. **2.0T EA888**：分层冷却缸体、集成排气歧管，暖机速度更快；高功率版本强化活塞、冷却系统，持续大负荷输出稳定，高速再加速储备充沛。
