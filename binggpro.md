# 五菱缤果Pro 动力参数介绍
全系**前置单永磁同步前驱电机**，统一65kW动力，分330km风冷、403km液冷两套续航版本，标配3.3kW V2L对外放电；整车4年/15万公里质保，纯电无燃油需求。

## 一、统一电机硬件参数
- 电机类型：前置水冷扁线永磁同步电机（12合一集成电驱）
- 峰值功率：65kW（88Ps）
- 峰值扭矩：110N·m
- 变速箱：电动车单速固定齿比减速器
- 最高车速：130km/h
- 0-50km/h加速约4.5s，城市起步轻快线性，无顿挫

## 二、330km 风冷续航版本
1. 电池规格：31.9kWh 风冷磷酸铁锂电池
2. CLTC纯电续航：330km
3. 补能速度
    - 直流快充30%-80%：35分钟（1C快充）
    - 家用慢充满电：8.9小时
4. 城市百公里电耗：10.5~12.3kWh
5. 适配场景：日常通勤、短途代步、市区接送

## 三、403km 液冷长续航版本
1. 电池规格：37.9kWh 液冷2.0磷酸铁锂电池，全温域温控，冬夏续航衰减更小
2. CLTC纯电续航：403km
3. 补能速度
    - 直流快充30%-80%：15分钟（3C高压快充）
    - 家用慢充满电：5.9小时
4. 城市百公里电耗：10.5~12.3kWh
5. 适配场景：一周一充、跨城短途出游、长途通勤

## 四、动力系统工作逻辑
1. 经济模式：限制电机输出，加大动能回收力度，降低电耗，适合匀速通勤；
2. 舒适模式：动力输出线性柔和，回收力度适中，兼顾平顺与能耗；
3. 运动模式：电机满功率释放，加速响应更快，超车、爬坡更从容；
4. 减速/下坡：电机反向制动发电储能，两档动能回收可调，强回收接近单踏板驾驶；
5. 全系标配3.3kW外放电，露营可带动电磁炉、投影仪等家用设备。
