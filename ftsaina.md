# 广汽丰田赛那SIENNA 动力系统完整参数
## 一、整车动力基础概况
- 车型定位：TNGA-K平台中大型7座混动MPV，全系仅提供**2.5L第五代THS II双擎混动**，分前驱、E-Four电子四驱两套驱动
- 燃油要求：加注92#无铅汽油，国六b排放标准
- 变速箱：E-CVT电控混动专用无级变速箱
- 油箱容积：68L
- 质保政策：整车3年/10万公里；混动动力电池8年/20万公里
- 驾驶模式：NORMAL标准/ECO经济/SNOW雪地，附带EV低速纯电模式
- 最高电子限速：180km/h
- 核心技术：A25D高热效率阿特金森发动机、VVT-iE电动可变气门、D-4S双喷射、E-Four后桥独立电机四驱、动能回收系统

## 二、2.5L混动发动机基础参数
- 发动机型号：A25D 直列四缸16气门全铝合金机身
- 排量：2487mL
- 循环形式：阿特金森循环，最高热效率41%
- 供油方式：D-4S缸内直喷+歧管双喷射
- 发动机最大功率：139kW（189Ps）/6000rpm
- 发动机峰值扭矩：236N·m / 3600–5200rpm
- 配气机构：进气VVT-iE电动可变气门、排气VVT-i液压可变气门

## 三、两套驱动版本电机与综合动力参数
### 1. 2.5L双擎 前置前驱版（家用主流）
- 前驱动电机：永磁同步电机
- 前电机峰值功率：134kW
- 前电机峰值扭矩：270N·m
- 系统综合最大功率：181kW（246Ps）
- 系统综合峰值扭矩：391N·m
- WLTC综合百公里油耗：5.66–5.77L
- 满油综合续航：1100km+
- 特点：结构简单、能耗更低，城市满载7人通勤油耗表现优异

### 2. 2.5L双擎 E-Four电子四驱版（雨雪全能）
- 电机布局：前134kW主电机 + 后桥独立40kW辅助电机
- 系统综合最大功率：181kW（246Ps）
- 系统综合峰值扭矩：391N·m
- WLTC综合百公里油耗：6.13L
- 四驱扭矩分配：前后轴100:0~20:80动态调节
- 特点：后桥电机独立驱动，无传动轴，冰雪、湿滑路面抑制打滑，轻度非铺装路面稳定性更强

## 四、E-CVT混动变速箱工作逻辑
1. **EV纯电模式**：低速挪车、拥堵蠕行，发动机熄火，电机单独驱动，零油耗低噪音
2. **低速混动**：电机为主输出，发动机按需启动辅助，动力平顺无顿挫
3. **中高速巡航**：发动机可直驱车轮，减少能量二次转换损耗，优化长途油耗
4. **急加速/满载爬坡**：发动机+前后电机同步叠加输出，提升爆发力
5. **减速/制动**：切断发动机动力，电机反向发电回收动能，减少刹车片磨损

## 五、E-Four电子四驱系统（四驱专属）
1. 结构原理：后桥搭载独立40kW永磁电机，无机械传动轴，电控独立控制后轮扭矩
2. 动态分配逻辑：日常巡航纯前驱节省能耗；起步、打滑、爬坡自动向后轮分配扭矩
3. 雪地模式：主动提升后轮动力输出，配合VSC车身稳定系统抑制车轮空转
4. 优势：轻量化结构，相比机械四驱油耗增幅极小，兼顾城市通勤与湿滑路面安全性

## 六、第五代THS混动能量管理系统
1. 智能油电分配：行车电脑实时根据车速、油门、负载自动切换驱动模式
2. 多级动能回收：滑行、下坡、刹车分梯度回收电能，充电效率更高
3. 电池恒温管理：锂离子混动电池搭载独立散热系统，高温散热、低温预热，延缓电芯衰减
4. 安全机制：碰撞瞬间切断高压回路，杜绝漏电风险

## 七、底盘、悬挂与制动配套
1. 悬架结构（全系统一）
    - 前悬架：麦弗逊式独立悬架，带横向稳定杆
    - 后悬架：E型多连杆独立悬架，舒适化调校，大幅过滤路面颠簸，抑制MPV车身侧倾
2. 转向系统：EPS电动助力转向，低速轻盈，5.7m最小转弯半径，大车城市通行灵活
3. 制动系统
    - 前后实心刹车盘，混动车型优先动能回收减速
    - 标配ABS/EBD、VSC车身稳定、TRC牵引力控制、上坡辅助HAC、自动驻车
4. 底盘配套：数字实时胎压监测、整车底盘防腐防护涂层

## 八、动力安全防护机制
- 发动机高温、高负荷分级限功率保护，满载长途动力不衰减
- 混动电池多重过压、欠压、过温电控保护，延长使用寿命
- 低重心电池布置，降低高速过弯侧倾，提升整车行驶稳定
- VSC车身稳定系统实时修正车身姿态，抑制加速打滑、弯道甩尾
