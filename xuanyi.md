# 2026款日产轩逸 动力参数介绍（前置前驱，1.6L经典燃油版、1.6L新款燃油版、e‑POWER超混电驱版）
## 一、整套动力架构概述
东风日产轩逸为四门五座前置前驱家用轿车，一共三套动力：1.6L经典版、1.6L新款高配燃油版、e‑POWER增程式电驱版；全部加注92#无铅汽油，无需高压充电。燃油版匹配CVT无级变速箱，e‑POWER采用1.2L增程器发电、电机单独驱动车轮，单速固定齿比变速箱。整车质保3年/10万公里，e‑POWER三电系统质保8年/15万公里。

### 1、1.6L经典版燃油动力（HR16发动机）
- 发动机型号：HR16‑DE 1.6L直列四缸自然吸气，双C‑VTC可变气门正时、铝合金缸体缸盖
- 最大功率：90kW（122Ps）/6000rpm
- 峰值扭矩：155N·m/4000rpm
- 变速箱：CVT无级变速箱
- 性能：0‑100km/h加速12.4s，最高车速183km/h，WLTC综合油耗5.99L/100km
- 油箱容积50L，满油续航约830km

### 2、1.6L新款高配燃油版（优化版HR16发动机）
- 发动机：升级e‑VTC电动可变气门正时，镜面熔射缸孔，降低发动机摩擦损耗
- 最大功率：99kW（135Ps）/6300rpm
- 峰值扭矩：159N·m/4000rpm
- 变速箱：CVT无级变速箱
- 性能：0‑100km/h加速12.1s，最高车速186km/h，WLTC综合油耗5.57L/100km
- 油箱容积47L，满油续航约785km

### 3、e‑POWER超混电驱版（增程式，1.2L三缸+前置驱动电机）
- 增程发动机：HR12 1.2L三缸自吸，仅用来发电，不直驱车轮，最大功率53kW（72Ps）
- 驱动电机：前置永磁同步电机，电机功率100kW（136Ps），峰值扭矩300N·m
- 传动形式：单速固定齿比变速箱，全程电机驱动，行驶质感接近纯电车
- 性能：0‑100km/h加速7.9s，WLTC综合油耗4.18L/100km
- 油箱容积41L，满油续航可达980‑1100km，不支持外接快充，仅依靠发动机发电+刹车滑行回收补能

### 四种驾驶模式
经济、标准、运动、ECO节能模式；e‑POWER额外拥有B挡强动能回收模式。

## 二、e‑POWER增程系统工作逻辑
1. **低速通勤模式**：发动机不启动，依靠电池储存电能，电机单独驱动；
2. **串联发电模式**：城市拥堵路况，1.2L增程器恒定在高效区间发电，电能供给电机；
3. **急加速模式**：增程器满负荷发电+电池同时放电，电机全力输出；
4. **滑行制动回收**：松油门、刹车时电机反向发电，能量回充至动力电池。

## 四、底盘与制动（动力配套部分）
1. 悬架结构：**前麦弗逊独立悬架+后扭力梁式非独立悬架**，底盘偏向舒适性调校，滤震柔和，无自适应减振。
2. 转向系统：EPS电动随速助力转向，低速轻盈，高速沉稳。
3. 制动系统：前通风盘、后实心盘刹车；e‑POWER版本带有动能回收制动，降低刹车片损耗；标配ABS、ESP车身稳定系统、上坡辅助、胎压监测。

## 五、e‑POWER三电防护说明
1. 电池采用小型风冷锂电池，布局在底盘下方；具备IP67防尘防水等级。
2. 搭载BMS电池管理系统，实时监控电芯电压、温度、压差；车辆发生碰撞后切断高压电路，实现过压、欠压、过热保护。
3. 电机电控集成化设计，散热稳定，城市长期低速工况动力输出稳定。
