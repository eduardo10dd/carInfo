# 2026款哈弗H6 动力参数介绍
提供**1.5T低功率燃油、1.5T高功率燃油、2.0T高功率燃油、Hi4插电混动四驱**四套动力，全系加注92#无铅汽油；整车3年/10万公里质保，插混三电系统8年/15万公里质保，插混标配3.3kW V2L对外放电。

## 一、四套动力分项硬件参数
### （一）1.5T低功率（GW4B15L）入门燃油版
1. 发动机参数
- 直列四缸涡轮增压，双VVT可变气门
- 最大功率：110kW（150Ps）/5600rpm
- 峰值扭矩：218N·m /1400–4000rpm
2. 变速箱：7速湿式双离合7DCT
3. 性能能耗
- 0‑100km/h加速：约10.6s
- WLTC综合油耗：7.13L/100km
- 油箱容积：58L

### （二）1.5T高功率（GW4G15M）家用主力燃油版
1. 发动机参数
- 1.5GDIT EVO，VGT可变截面涡轮、350Bar直喷
- 最大功率：135kW（184Ps）/5500rpm
- 峰值扭矩：275N·m /1500–4000rpm，1500rpm输出满扭矩
2. 变速箱：7速湿式双离合7DCT
3. 性能能耗
- 0‑100km/h加速：约9.2s
- WLTC综合油耗：7.52L/100km
- 油箱容积：58L

### （三）2.0T高功率（GW4N20A）性能燃油版
1. 发动机参数
- 双流道低惯量涡轮、双平衡轴
- 最大功率：175kW（238Ps）/5000rpm
- 峰值扭矩：385N·m /1800–4000rpm
2. 变速箱：9速湿式双离合9DCT，可选适时四驱
3. 性能能耗
- 0‑100km/h加速：7.8s；四驱版8.2s
- WLTC综合油耗：两驱8.1L，四驱8.7L
- 油箱容积：58L

### （四）Hi4 1.5T插混四驱（三擎九模电混）
1. 混动专用1.5T阿特金森发动机
- 发动机最大功率：115kW，峰值扭矩240N·m，热效率38.5%
2. 前后双电机四驱
- 前电机：130kW；后电机：150kW
- 系统综合最大功率：245kW，综合峰值扭矩595N·m
3. 变速箱：专用混动DHT，三擎九模智能切换
4. 动力电池：19.09kWh磷酸铁锂液冷电池
5. 性能能耗续航
- CLTC纯电续航85km，满油满电综合续航1100km
- 0‑100km/h加速：6.8s，顶配高性能版4.8s
- WLTC馈电综合油耗：5.3L/100km
- 快充30%-80%约18分钟，标配3.3kW V2L外放电
- 油箱容积：52L

## 二、四套动力工作逻辑
1. **1.5T低功率**：入门代步，维保成本低，适合城市短途通勤、年均里程少用户。
2. **1.5T高功率**：家用均衡款，低扭充沛，城市超车轻快，兼顾动力与燃油经济性。
3. **2.0T高功率**：高功率性能取向，满载、高速再加速储备充足；适时四驱雨雪、轻度烂路提升抓地力。
4. **Hi4插混四驱**
    - 城市短途：后桥电机单独纯电驱动，零油耗；
    - 匀速高速：后桥电机解耦，仅前轴工作降低拖拽损耗；
    - 急加速/爬坡/湿滑路面：前后电机+发动机并联输出，四驱瞬时锁止；
    - 减速下坡：电机反向发电储能，两档动能回收可调；
    - 支持户外大功率放电，露营可带动全套家电。
