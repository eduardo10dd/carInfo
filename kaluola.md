# 2026款一汽丰田卡罗拉 动力技术参数介绍（三套燃油 + THS非插混双擎 + 双擎E+插混）
## 一、整车基础概况
卡罗拉定位紧凑型家用前驱轿车，车身尺寸4635×1780×1435mm，轴距2700mm；全系前置前驱，统一加注92#无铅汽油；分为**1.5L三缸自吸、1.2T四缸涡轮、1.8L THS IV双擎（非插混）、卡罗拉双擎E+插混PHEV**四大动力；低配1.5L采用前麦弗逊+后扭力梁，1.2T/双擎标配前麦弗逊+后双横臂独立悬架；主打平顺省油、低维保成本、高可靠性。

## 第一部分：燃油动力两套总成
### 1. 1.5L M15B 三缸自然吸气（入门经济版）
- 发动机：1490mL直列三缸，D-4直喷、VVT-iW广角可变气门，压缩比12.9:1
- 最大功率：89kW（121Ps）/6500rpm
- 峰值扭矩：148N·m /4600–5000rpm
- 变速箱：S-CVT模拟10速无级变速箱
- 0-100km/h加速：11.7s
- 最高电子限速：180km/h
- WLTC综合油耗：5.41L/100km
- 油箱容积50L，满油综合续航约920km
- 悬架：前麦弗逊+后扭力梁非独立悬架
- 适用场景：城市短途代步、预算有限家用，追求低购车成本

### 2. 1.2T 8NR-FTS 四缸涡轮增压（家用走量燃油）
- 发动机：1197mL直列四缸水冷涡轮，双喷射、低惯量涡轮，阿特金森循环
- 最大功率：85kW（116Ps）/5200–5600rpm
- 峰值扭矩：185N·m /1500–4000rpm，1500转即可输出最大扭矩，高速超车动力更充沛
- 变速箱：S-CVT模拟10速无级变速箱
- 0-100km/h加速：10.4s
- 最高电子限速：180km/h
- WLTC综合油耗：5.88L/100km
- 油箱容积50L，满油综合续航约850km
- 悬架：前麦弗逊+后双横臂独立悬架
- 适用场景：经常高速、满载出行，想要更强中段加速与更好滤震

## 第二部分：1.8L THS IV 智能电混双擎（非插混，无需外接充电）
### 1. 8ZRFXE 1.8L阿特金森循环发动机
- 排量1798mL，高压缩比13:1，仅发电+高速直驱
- 发动机最大功率：72kW（98Ps）/5200rpm
- 发动机峰值扭矩：142N·m /3600rpm

### 2. 前置永磁同步驱动电机
- 电机峰值功率：70kW（95Ps）
- 电机峰值扭矩：185N·m，起步纯电输出，低速零抖动
- 变速箱：THS专用E-CVT行星齿轮电子无级变速，无换挡顿挫

### 整套系统综合性能
- 系统综合峰值功率：101kW（137Ps）
- 0-100km/h加速：10.2s
- 最高电子限速：180km/h
- WLTC综合油耗：4.06–4.28L/100km，市区通勤实测低至3.8L
- 储能电池：1.3kWh小型镍氢电池，闪充闪放，**无外接充电接口**
- 油箱容积50L，满油综合续航超1100km
- 悬架：前麦弗逊+后双横臂独立悬架

### THS混动四大工作模式
1. **EV纯电模式**：低速、电量充足时发动机停机，电机单独驱动，零油耗、静谧；
2. **串联发电模式**：低速低电量，1.8L发动机稳定高效区间发电供给电机；
3. **高速直驱模式**：高速巡航发动机直接驱动车轮，减少电能转换损耗；
4. **滑行/制动能量回收**：松油门、刹车电机反向发电，电能存入电池循环利用。

## 第三部分：卡罗拉双擎E+ 插电混动PHEV（可上绿牌）
### 动力总成架构：1.8L阿特金森+双电机插混系统
- 发动机参数与普通双擎一致：72kW/142N·m
- 主驱动电机峰值功率：53kW，峰值扭矩207N·m
- 系统综合输出功率：100kW（136Ps）
- 变速箱：E-CVT混动专用电子无级变速

### 电池、续航与补能规格
- 电池：10.5kWh松下三元锂电池，液冷温控
- CLTC纯电续航：60km，日常通勤可纯电零油耗行驶
- NEDC综合油耗：1.3L/100km；亏电WLTC油耗约4.3L，亏电仍保持双擎同级低油耗
- 油箱容积50L，满油满电综合续航985km+
- 充电规格：仅支持交流慢充，家用220V便携充5.5小时充满，公共充电桩约3.5小时；无直流快充
- V2L外放电：不支持

### PHEV工作模式
1. EV纯电优先模式：起步默认纯电，电量耗尽自动切换混动；
2. 混动均衡模式：油电协同，兼顾动力与油耗；
3. 电量保持模式：强制发动机发电，保留电量用于市区短途纯电；
4. 能量回收模式：滑行、制动回收电能存入电池。

## 五、动能回收与制动系统
1. **燃油版（1.5L/1.2T）**：无强能量回收，制动踏板线性柔和；四轮盘式制动，标配ESC车身稳定、上坡辅助、牵引力控制；无单踏板模式。
2. **THS双擎/双擎E+混动**：多级可调能量回收强度，长下坡自动提升回收功率，依靠电机回收减速，大幅降低刹车片磨损；混动专属机电协同制动，踏板脚感均匀不突兀。

## 六、底盘硬件配置
1. 悬架区分：1.5L低配后扭力梁，滤震一般、成本更低；1.2T/双擎全系升级后双横臂独立悬架，过弯侧倾更小，烂路颠簸过滤更细腻。
2. 转向：电动助力转向，低速轻盈、高速沉稳，紧凑型车身转弯半径小，城市停车灵活。
3. 车身：高刚性笼式车身，标配10安全气囊、TSS 3.0智行安全系统。

## 七、动力安全防护机制
1. 全系发动机水冷循环温控，长时间拥堵、满载爬坡分级限制输出功率，避免高温损伤；涡轮版熄火延时散热，降低热衰减。
2. CVT/E-CVT变速箱内置油冷散热，长期城市蠕行不易触发过热保护，耐久性稳定。
3. 混动电池包IP68防尘防水，24小时BMS监测电芯电压、温度，碰撞切断高压回路；镍氢/三元电芯均通过针刺、挤压安全测试。
4. 多重电控分级保护：爆震、超温、油压异常自动限制动力输出；ESC稳定系统抑制湿滑路面打滑、推头甩尾。
