# 2026款极氪001 动力参数介绍
全系标配**900V碳化硅高压电驱平台**，分**后置单电机后驱、前后双电机四驱**两套动力，提供95kWh磷酸铁锂、103kWh三元锂两种电池，全系支持6kW V2L对外放电；整车4年/15万公里质保，三电系统首任非营运终身质保。

## 一、动力硬件分项参数
### （一）单电机后驱版（103kWh三元锂）
1. 驱动电机：后置永磁同步油冷电机
- 峰值功率：370kW（503Ps）
- 峰值扭矩：535N·m
2. 电池：103kWh三元锂液冷电池包
- CLTC纯电续航：810km
- 快充：10%-80%约10分钟，峰值680kW高压快充
3. 性能数据
- 0‑100km/h加速：4.98s
- 电子限速最高车速：280km/h
4. 电驱特性：单电机高效取向，日常通勤电耗更低，兼顾续航与够用性能

### （二）双电机四驱版本（两套电池）
双电机采用**前异步感应+后永磁同步**组合，兼顾高速性能与低速能效
#### 1、95kWh磷酸铁锂四驱Max
- 前电机：240kW；后电机：440kW
- 系统综合最大功率：680kW（925Ps），综合峰值扭矩913N·m
- CLTC续航：710km
- 0‑100km/h加速：3.08s
- 快充10%-80%仅7分钟

#### 2、103kWh三元锂四驱Ultra（性能顶配）
- 电机功率、扭矩与95kWh四驱一致：综合680kW / 913N·m
- CLTC续航：730km
- 0‑100km/h加速：2.83s（全系最快）
- 快充10%-80%约10分钟

### 三电通用技术
1. 900V碳化硅电控：电能损耗降低15%，大功率快充温升控制优秀；
2. 全域液冷温控电池，带电池预加热，改善冬季续航衰减；
3. 全系标配6kW V2L户外外放电，支持露营全套电器；
4. 固定齿比电驱动，无变速箱换挡顿挫，响应毫秒级。

## 二、电驱工作逻辑
1. **后驱单电机**：城市通勤低负载高效省电，高速巡航持续稳定输出；经济模式限制功率降低电耗，运动模式释放全部扭矩。
2. **双电机智能四驱**
    - 匀速巡航：以后置永磁电机为主驱动，前电机低负荷待机，兼顾省电；
    - 急加速/爬坡/湿滑路面：前后电机同步满功率输出，毫秒级分配四轮扭矩，抑制打滑、推头；
    - 高速工况：前异步电机介入，高转速区间动力储备更强，极速可达280km/h。
3. **动能回收**：两档回收强度可调，强回收单踏板模式，减速、下坡反向发电储能，减少刹车片磨损。
