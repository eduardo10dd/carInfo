# 五菱星光730新能源 动力技术参数介绍（插混PHEV + 纯电EV双架构）
## 一、整车基础概况
星光730新能源定位家用7座中型MPV，车身尺寸4910×1850×1770mm，轴距2910mm，2+2+3七座布局；全系前置前驱，基于天舆架构打造，分为**灵犀3.0插电混动、纯电EV**两套新能源动力；插混加注92#无铅汽油，全系标配220V V2L对外放电、液冷磷酸铁锂神炼电池；底盘前麦弗逊独立悬架+后多连杆独立悬架，主打家用平顺、低用车成本。

## 第一部分：灵犀动力3.0 插电混动PHEV（125km纯电续航）
### 1. 1.5L阿特金森混动专用自吸发动机
- 型号：LBG 1485mL直列四缸自然吸气
- 发动机最大功率：78kW（106Ps）/5600rpm
- 峰值扭矩：130N·m /4000rpm
- 最高热效率43.2%，仅负责发电+高速直驱，多重NVH减振，增程介入抖动微弱
- 油箱容积52L

### 2. 前置永磁同步驱动电机（10合1集成电驱单元）
- 电机峰值功率：145kW（197Ps）
- 电机峰值扭矩：230N·m，低速起步满扭矩输出，城市超车轻快
- 变速箱：E-CVT电子无级变速，油电切换无顿挫

### 3. 整套系统综合性能
- 系统综合峰值功率：197kW
- 官方0-100km/h加速：9.2s
- 最高电子限速：170km/h
- WLTC综合油耗：1.83L/100km；WLTC亏电油耗：5.3L/100km
- CLTC纯电续航：125km，日常通勤可纯电行驶
- CLTC满油满电综合续航：1100km

### 4. 电池与补能规格
- 电池：20.5kWh磷酸铁锂神炼液冷电池，IP68防尘防水
- 直流快充：30%-80%约22分钟
- 6.6kW交流慢充，0-100%满充约3.2h
- V2L外放电：220V/3.3kW，支持露营炊具、户外工具

### 插混五大工作模式
1. **EV纯电模式**：电量充足发动机停机，电机单独驱动，零油耗、座舱静谧；
2. **串联增程模式**：低速、低电量工况，1.5L发动机稳定高效区间发电供给电机；
3. **高速直驱模式**：高速巡航锁止E-CVT，发动机直驱车轮，降低电能转换损耗；
4. **并联全力输出**：急加速、满载爬坡，发动机发电+电池大功率放电同步输出峰值动力；
5. **滑行/制动能量回收**：松油门、刹车电机反向发电，电能存入电池循环利用。

## 四、神炼电池全域液冷热管理与安全
1. 全系磷酸铁锂神炼电池，一体化液冷温控，搭配热泵空调，-10℃低温续航达成率75%以上；
2. 电池包IP68防尘防水，底部高强度钢制护板，通过针刺、挤压、火烧极限安全测试；
3. 24小时AI实时监测电芯电压、温度、压差，碰撞毫秒级切断全部高压回路；
4. 五横七纵高强度电池框架，降低托底撞击热失控风险。

## 五、动能回收与机电协同制动系统
1. 三级可调能量回收：舒适/标准/强回收；城市强回收可接近单踏板行驶，高速模式回收减弱，滑行距离更长；
2. 液压制动与电机能量回收协同工作，90%日常减速依靠电机反向发电储能，大幅减少刹车片磨损；
3. 坡道识别自动匹配回收力度，长下坡稳定车速同步补电，抑制刹车热衰减。

## 六、底盘硬件与动力安全防护机制
1. 悬架：前麦弗逊独立悬架（液压衬套）+后多连杆独立悬架，家用舒适调校，过滤细碎颠簸；全系电动助力转向，低速轻盈、城市停车灵活；
2. 插混1.5L发动机多级水冷温控，长时间馈电满载爬坡分级限制输出功率；
3. 前后电机独立水冷散热，多人满载长途行驶无热衰减；高压线束分层绝缘密封，高压故障毫秒级断电保护；
4. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；标配ESC车身稳定系统，抑制湿滑路面打滑、车身失控。
