# 2026款吉利银河M9 动力参数介绍
全系搭载**雷神EM-P AI电混2.0插混系统、3挡DHT Pro混动专用变速箱、800V高压碳化硅平台、神盾磷酸铁锂刀片电池**，加注92#无铅汽油；整车4年/15万公里质保，三电系统首任非营运终身质保，标配3.3kW V2L对外放电，分**前置单电机两驱、三电机分布式四驱**两套动力。

## 一、统一混动发动机基础参数（全系通用）
- 发动机型号：BHE15-CFZ 1.5T米勒循环四缸涡轮混动专用机
- 最大功率：120kW（163Ps）/5500rpm
- 峰值扭矩：255N·m /1800–4000rpm
- 核心技术：350Bar高压直喷、水冷EGR、低摩擦涂层，最高热效率44.26%~46.5%
- 作用：低速主要发电，高速可直驱车轮，兼顾高负荷稳定输出与低油耗
- 油箱容积：62L

## 二、两驱单电机动力总成（100km / 230km纯电续航）
### 1、电驱硬件
- 前置永磁同步驱动电机：峰值180kW / 350N·m
- 系统综合最大功率：300kW（408Ps），综合峰值扭矩605N·m
- 变速箱：3挡DHT Pro混动专用变速箱
### 2、两套电池版本
1. **18.4kWh 短续航版（CLTC纯电100km）**
    CLTC满油满电综合续航1325km，30%-80%快充约15分钟，WLTC馈电油耗5.7L/100km，0‑100km/h加速7.7s
2. **41.46kWh 长续航版（CLTC纯电230km）**
    CLTC满油满电综合续航1505km，30%-80%快充约15分钟，WLTC馈电油耗4.8L/100km，0‑100km/h加速7.3s

## 三、三电机四驱顶配（CLTC纯电210km）
### 1、分布式三电机布局（前180kW + 后双电机合计340kW）
- 前电机：180kW / 350N·m；后桥双电机合计340kW / 560N·m
- 系统综合最大功率：640kW（870Ps），综合峰值扭矩1165N·m
- 变速箱：同款3挡DHT Pro混动变速箱
### 2、电池与性能能耗
- 电池：41.46kWh宁德时代神盾磷酸铁锂液冷电池
- CLTC纯电续航210km，满油满电综合续航1300km
- WLTC馈电综合油耗5.5L/100km，0‑100km/h官方加速4.5s
- 800V高压快充，30%-80%仅15分钟，支持大功率户外放电

## 四、雷神EM-P混动系统工作逻辑
1. **城市低速拥堵**：纯电机驱动，发动机不启动，零油耗、无顿挫，静谧平顺；
2. **中低速工况**：1挡电驱为主，发动机按需发电，油电无缝切换；
3. **高速巡航**：切换3挡发动机直驱车轮，减少能量转换损耗，高速亏电动力无衰减；
4. **急加速/满载爬坡**：发动机+全部电机并联满功率输出，瞬时扭矩充沛；
5. **减速/下坡**：电机反向发电储能，两档动能回收可调，降低刹车片磨损；
6. **四驱专属扭矩矢量**：毫秒级分配前后轮动力，湿滑、交叉轴、陡坡脱困能力更强。
