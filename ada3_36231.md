# 2026款奥迪A3动力参数介绍（两厢Sportback / 三厢A3L Limousine，前置前驱豪华紧凑型轿车）
## 一、动力总成整体概述
2026款奥迪A3全系淘汰1.4T机型，统一搭载**EA211 1.5T EVO II米勒循环涡轮增压发动机**，部分版本搭载48V MHEV轻混系统，匹配7速S-tronic双离合（DQ200干式），前置前驱布局；全系支持92#无铅汽油，油箱容积51L，整车质保3年/10万公里。主打城市通勤、家用代步、精致豪华入门出行场景。

### 1、1.5T EVO II 发动机核心硬件
- 型号：DSV/DYZ 1.5L直列四缸涡轮增压
- 最大功率：118kW（160Ps）/5500rpm
- 峰值扭矩：250N·m / 1750–4000rpm
- 核心技术：
  1. VTG可变截面涡轮，大幅削弱低速涡轮迟滞；
  2. ACT主动闭缸技术，匀速巡航两缸运行，降低油耗；
  3. 米勒循环+电控冷却系统，热效率提升；
  4. 48V轻混版配备BSG发电机，提供起步扭矩补偿。
- 变速箱：7速S-tronic干式双离合DQ200，换挡响应快、传动效率高。

### 2、分版本性能、能耗区分
#### （1）标准版 1.5T 无48V轻混
- 适用：A3L三厢、Sportback两厢入门车型
- 0‑100km/h加速：8.7s（三厢A3L）/8.6s（两厢Sportback）
- 最高车速：200km/h
- WLTC综合油耗：三厢5.92L/100km；两厢6.04L/100km

#### （2）48V MHEV轻混高配版
- 轻混电机辅助功率10kW，起步、滑行能量回收，优化启停平顺性
- 加速成绩与普通版一致，综合油耗降低0.1–0.2L/100km
- WLTC综合油耗低至5.75L/100km，怠速启停更安静无抖动

### 驾驶模式
舒适、自动、动态、经济四种模式；S挡拉高转速，变速箱降挡更激进，油门响应更灵敏；手动换挡拨片全系可选。

## 二、1.5T+48V轻混工作逻辑
1. **低速城市通勤**：低负载自动切换两缸闭缸模式，48V电机辅助启停，消除怠速抖动，涡轮VTG快速介入，起步轻快不肉；
2. **匀速高速巡航**：四缸经济工况为主，轻混系统回收滑行动能储存，减少发动机负荷，压低长途油耗；
3. **急加速/超车**：立刻切换四缸全功率输出，VTG涡轮同步增压，48V电机瞬时补扭矩，动力衔接无迟滞；
4. **减速滑行**：松油门发动机关闭气缸，电机反向发电储能，减少刹车磨损，提升续航里程。

## 四、底盘与制动配套硬件
1. 悬架结构：全系**前麦弗逊独立悬架+后扭力梁非独立悬架**，德系偏紧致调校，滤震干脆，高速侧向支撑充足；A3L长轴版轴距2730mm，乘坐舒适性优化。
2. 通过性参数：最小转弯半径5.4m，城市狭窄道路、小区掉头灵活；轮胎规格205/55 R16、225/45 R17高低配区分。
3. 转向制动：EPS电动随速助力转向；前通风盘+后实心刹车盘；标配ABS、ESP车身稳定、上坡辅助、胎压数字显示。

## 五、48V轻混系统功能说明
1. 平顺启停：等红灯、低速蠕行平稳熄火启动，消除传统燃油车启停抖动；
2. 扭矩补偿：起步、急加速瞬间电机输出辅助扭矩，弥补涡轮迟滞，低速动力响应更快；
3. 动能回收：滑行、制动回收动能存入48V电池，为车载电器供电，降低发动机负载；
4. 巡航优化：匀速行驶减少发动机喷油负荷，小幅降低综合工况油耗。
