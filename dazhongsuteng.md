# 2026款一汽-大众速腾 动力参数介绍
## 一、动力总览
全系前置前驱布局，当前主力在售为**300TSI 1.5T EVO II**单一动力；老款280TSI 1.4T为停售库存版本，参数一并对比；全系匹配7速干式双离合7DCT，加注92#无铅汽油，最高电子限速200km/h；驾驶模式：经济/标准/运动/个性化。

## 二、300TSI 1.5T EVO II（当前全系主力）
### 1. 发动机核心参数
- 型号：EA211-DSV/DYZ 直列四缸涡轮增压
- 排量：1498mL
- 进气循环：米勒循环+VTG可变截面涡轮、350bar高压直喷、冷却EGR、等离子涂层气缸壁
- 最大功率：118kW（160Ps）/5500rpm
- 峰值扭矩：250N·m /1750–4000rpm
- 关键优势：1500rpm即可输出80%扭矩，涡轮迟滞大幅削弱；扭矩平台拓宽至4000转，高速再加速储备更强

### 2. 变速箱：7速干式双离合7DCT
1. 传动效率高，轻量化结构，城市通勤油耗表现优异
2. 预换挡逻辑，升降挡衔接平顺；支持手动换挡模式、运动换挡逻辑
3. 智能温控散热，日常家用工况稳定；拥堵蠕行工况会适当限制扭矩保护离合器

### 3. 性能与能耗
- 官方0-100km/h加速：8.7s
- WLTC综合工况油耗：5.49–5.77L/100km
- 油箱容积50L，满油综合续航约780–820km

## 三、280TSI 1.4T EA211（老款停售库存）
### 1. 发动机参数
- 型号：EA211-DSB 直列四缸涡轮增压，奥托循环、双流道涡轮
- 排量：1395mL
- 最大功率：110kW（150Ps）/5000–6000rpm
- 峰值扭矩：250N·m /1750–3000rpm
### 2. 性能能耗
- 0-100km/h官方加速：9.2s
- WLTC综合油耗：5.96L/100km
- 仅推荐加注95#汽油，用车成本高于1.5T版本

## 五、变速箱换挡逻辑
1. **经济模式**：升挡积极，压低巡航转速，优先降低燃油消耗，松油门拖拽感轻微
2. **标准模式**：均衡换挡节奏，城市跟车、高速巡航兼顾平顺与动力响应
3. **运动模式**：延迟升挡，拉高转速，深踩油门连续多挡快速降挡，动力输出更直接
4. **个性化模式**：可单独自定义油门灵敏度、变速箱换挡逻辑、转向力度

## 六、动力安全防护机制
1. 1.5T发动机智能多级温控系统，高温爬坡、长时间拥堵分级限制输出功率；熄火后涡轮延时水冷散热，延长涡轮寿命
2. 7DCT变速箱实时监测离合器温度，重度拥堵自动限制扭矩输出，避免离合器过热磨损
3. ESC车身稳定系统实时修正车身姿态，抑制起步打滑、高速横风偏移、弯道推头
4. 多重电控容错：发动机爆震保护、涡轮超压保护、冷却系统故障限功率输出
