# 2026款方程豹豹5 动力技术参数介绍（DMO插混越野平台）
## 一、整车基础概况
豹5基于**DMO超级混动越野平台**打造，定位中大型插混硬派越野SUV，采用非承载式梯形大梁车身，5门5座布局；全系纵置1.5T插混+前后双电机智能电四驱，统一加注92#无铅汽油；分为125km标准续航、210km长续航两大版本，高配搭载云辇-P智能液压车身；全系标配6kW大功率V2L对外放电，首任非营运车主三电系统终身质保；底盘标配前后双叉臂独立悬架，标配中央电控锁+后桥电控差速锁，顶配增加前差速锁实现三把锁。

## 二、DMO混动动力总成（全系通用动力硬件）
### 1. 1.5T纵置高功率混动专用发动机
- 排量：1498mL直列四缸涡轮增压，米勒循环，最高热效率46.06%
- 发动机最大功率：143kW（194Ps）/5500rpm
- 发动机峰值扭矩：273N·m /1800–4000rpm
- 定位：稳定高效发电+高速直驱，纵置布局适配硬派大梁底盘，长爬坡、高温工况动力输出稳定
- 油箱容积75L

### 2. 前后双水冷永磁同步驱动电机
- 前驱动电机峰值功率：200kW，峰值扭矩350N·m
- 后驱动电机峰值功率：285kW，峰值扭矩410N·m
- 整套系统综合峰值功率：505kW（687匹）
- 系统综合峰值扭矩：760N·m，轮端等效扭矩可达8000N·m，极限攀爬、脱困动力储备充足
- 变速箱：纵置专用EHS电子无级E-CVT，油电切换无顿挫，保留完整机械四驱传动链路，纯电行驶不脱开四驱结构
- 官方0-100km/h加速：4.8s
- 最高电子限速：180km/h

### DMO五大工作模式
1. **EV纯电模式**：电量充足发动机停机，双电机四驱驱动，城市通勤零油耗、静谧性拉满，四驱结构全程联动；
2. **串联增程模式**：低速、低电量、越野工况，1.5T发动机稳定高效区间发电，电能供给前后电机输出大扭矩；
3. **高速直驱模式**：高速巡航发动机锁止直驱车轮，减少电能转换损耗，降低长途能耗；
4. **并联全力输出模式**：急加速、45°陡坡攀爬、重度越野，发动机满负荷发电+电池大功率放电同步释放760N·m综合扭矩；
5. **滑行/制动能量回收**：松油门、刹车电机反向发电，最大回收功率100kW，电能存入刀片电池循环利用。

## 三、两套刀片电池、续航与补能规格
### 1. 125km标准续航版（31.8kWh一代刀片电池）
- 电池规格：31.8kWh磷酸铁锂刀片电池，液冷直热温控
- CLTC纯电续航：125km，WLTC纯电续航100km
- WLTC综合油耗：1.79L/100km，NEDC亏电油耗7.8L/100km
- CLTC满油满电综合续航：1015km
- 直流快充峰值100kW，30%-80%电量约16分钟
- 6.6kW交流慢充，0-100%满充约5h

### 2. 210km长续航/云辇闪充版（47.8kWh二代刀片电池）
- 电池规格：47.8kWh大容量二代刀片电池，800V高压闪充平台
- CLTC纯电续航：210km，WLTC纯电续航160km
- WLTC综合油耗：1.23L/100km，NEDC亏电油耗7.8L/100km
- 普通长续航版综合续航1015km；云辇闪充版CLTC综合续航1310km
- 快充区分：
  - 普通210km版：100kW直流快充，30%-80%约16分钟
  - 云辇闪充版：150kW高压闪充，30%-80%仅9分钟
- 6kW V2L大功率对外放电，支持户外露营、车载设备供电
- 支持20kW原地外接补电，驻车户外可快速补能

## 五、四驱越野核心系统
1. **电控三把锁**：中央能量锁+后桥电控牙嵌锁，顶配标配前差速锁，毫秒级锁止响应，单轮着地即可脱困，最大爬坡度45°；
2. **4L低速四驱挡**：扭矩放大2.5倍，岩石、陡坡攀爬扭矩进一步放大；
3. 多地形模式：自动/雪地/泥沙/岩石/运动/暴力/拖车/豹式掉头八大模式；标配CCO蠕行、TAB豹式掉头、陡坡缓降；
4. 通过性参数：普通版离地间隙220mm，接近角35°、离去角32°；云辇-P车身高度60mm可调，最高离地间隙310mm，接近角39°、离去角35°，最大涉水深度700mm。

## 六、刀片电池全域液冷热管理与安全
1. CTC电池车身一体化，电池嵌入大梁中间，整车扭转刚度大幅提升，电池底部配备2mm加厚钢制全包护板；
2. 全域直冷直热一体化热泵热管理，回收电机、发动机余热预热电池，-10℃低温纯电续航达成率72%以上；
3. 电池包IP68防尘防水，通过针刺、挤压、火烧极限安全测试，无热扩散风险；
4. 24小时AI-BMS全域电芯监测，实时采集电压、温度、压差，碰撞2毫秒级切断全部高压回路。

## 七、动能回收与机电协同制动系统
1. 三级可调能量回收强度：舒适/标准/越野；越野模式回收减弱，滑行脱困更灵活；城市强回收模式接近单踏板行驶；
2. IPB线控制动CRBS机电协同系统，90%日常减速依靠电机反向发电储能，大幅降低刹车片磨损；
3. 长下坡依靠能量回收稳定车速，配合低速四驱，抑制连续山路刹车热衰减。

## 八、底盘硬件与三电安全防护机制
1. 悬架：全系前后双叉臂独立悬架，液压衬套抑制颠簸；云辇-P版本支持车身升降、连续阻尼可调，兼顾铺装路舒适与越野通过性；电动助力转向，高速沉稳、低速轻盈；
2. 1.5T混动发动机多级水冷温控，长时间馈电满载爬坡、高温越野分级限制输出功率，涡轮熄火延时散热；
3. 前后双电机独立水冷油冷散热，连续激烈越野无热衰减；800V高压线束分层绝缘密封，高压故障毫秒级断电保护；
4. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；ESC车身稳定系统联动四驱扭矩分配，抑制碎石、雪地打滑甩尾。
