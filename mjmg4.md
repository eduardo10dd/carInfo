# 2026款名爵MG4 动力技术参数介绍（前驱家用版+XPOWER四驱性能版）
## 一、整车基础概况
MG4定位纯电紧凑型五门两厢轿车，车身尺寸4395×1842×1551mm，轴距2750mm；分为**前置前驱家用版、XPOWER前后双电机四驱性能版**两套动力；家用版提供437km磷酸铁锂、530km半固态两套电池；全系搭载CTB电池车身一体化、HeatMatrix一体式液冷热管理、3.3kW V2L对外放电；底盘前麦弗逊独立悬架+后变截面扭力梁非独立悬架；整车3年/10万公里，三电8年/15万公里，半固态版本可选三电终身质保。

- 0-50km/h加速：3.0s，短途代步响应迅猛
- 官方0-100km/h加速：8.1s
- 最高电子限速：160km/h

- 驾驶模式：经济/标准/运动/雪地/自定义5种动力模式，可调动力输出与动能回收强度

## 四、CTB魔方电池全域液冷热管理与安全
1. CTB电池车身一体化技术，电池上盖与车身底板融合，整车扭转刚度31000N·m/deg，碰撞安全性大幅提升；
2. HeatMatrix一体式液冷直冷直热系统，回收电机余热加热电池，兼顾充电速度与低温续航；
3. 电池包IP68防尘防水，底部钢制全包防护板，通过火烧、挤压、针刺、浸水八项极限安全测试；
4. 24小时AI-BMS全域电芯监测，实时采集电压、温度、压差，碰撞毫秒级切断全部高压回路。

## 五、动能回收与机电协同制动系统
1. 多级可调动能回收，支持单踏板模式；经济模式强回收，减少刹车片损耗；运动/雪地模式回收减弱，滑行距离更长；
2. IBS集成式机电协同制动，90%日常减速依靠电机反向发电储能，续航提升约16%；
3. 制动硬件：前通风盘、后实心盘制动，XPOWER四驱升级高性能制动卡钳，100-0km/h制动距离更短；标配上坡辅助、陡坡缓降、VMC车身稳定控制系统。

## 六、底盘硬件与三电安全防护机制
1. 悬架：全系前麦弗逊独立悬架（液压衬套）+后变截面扭力梁非独立悬架；中欧联合底盘调校，兼顾城市舒适与高速稳定；5.3m小转弯半径，城市通勤泊车灵活；
2. 前驱六合一水冷电机、XPOWER双电机独立油冷散热，长时间满载、连续激烈驾驶无热衰减；电控集成碳化硅模块，降低高压电能损耗；
3. 400V高压线束分层绝缘密封，高压故障毫秒级断电保护；多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；
4. XPOWER四驱联动ESC车身稳定系统，雨雪、碎石路面动态分配前后扭矩，抑制推头、甩尾失控。
