# 2026款上汽大众途观L 动力参数介绍
## 一、动力总览
全系三套燃油动力：**300TSI 1.5T EVO II、330TSI 2.0T低功率、380TSI 2.0T高功率**；380TSI提供两驱/4MOTION智能四驱，全系匹配**DQ381 7速湿式双离合**，加注95#汽油，国六b排放；最高电子限速200km/h；驾驶模式：经济/标准/运动/个性化。

## 二、三套发动机详细参数
### 1. 300TSI 1.5T EVO II（入门家用省油版）
- 发动机型号：EA211-DSV 直列四缸涡轮增压
- 排量：1498mL，米勒循环+VTG可变截面涡轮、350bar高压直喷、冷却EGR
- 最大功率：118kW（160Ps）/5500rpm
- 峰值扭矩：250N·m /1750–4000rpm
- 官方0-100km/h加速：9.9s
- WLTC综合油耗：6.38L/100km
- 适配场景：城市通勤、日常代步，注重燃油经济性

### 2. 330TSI 2.0T低功率（主力家用两驱）
- 发动机型号：EA888-DPL 第三代EA888直列四缸涡轮
- 排量：1984mL，双流道涡轮、可变气门正时、200bar直喷
- 最大功率：137kW（186Ps）/4100–6000rpm
- 峰值扭矩：320N·m /1500–4000rpm，1500转即可输出满扭矩，低扭充沛
- 官方0-100km/h加速：8.7s
- WLTC综合油耗：7.27L/100km
- 仅提供前置前驱，满载5人、高速超车动力储备充足

### 3. 380TSI 2.0T高功率（性能/四驱版）
- 发动机型号：EA888-DTH 高功率调校EA888
- 排量：1984mL，强化涡轮冷却、电子废气旁通阀
- 最大功率：162kW（220Ps）/4500–6200rpm
- 峰值扭矩：350N·m /1600–4300rpm，扭矩平台更宽，高速再加速爆发力更强
- 官方0-100km/h加速：7.5s
- WLTC综合油耗：两驱7.52L / 四驱7.8L
- 可选前置前驱、4MOTION智能适时四驱，雨雪、轻度烂路稳定性大幅提升

## 三、DQ381 7速湿式双离合变速箱（全系通用）
1. 湿式结构，离合器总成浸泡变速箱油循环散热，拥堵蠕行不易高温保护，可靠性优于干式DQ200
2. 预换挡逻辑，升降挡衔接平顺；支持手动换挡模式、运动换挡逻辑
3. 多模式换挡策略：
    - 经济模式：升挡积极，压低巡航转速，降低油耗
    - 运动模式：延迟升挡、拉高转速，深踩油门连续多挡快速降挡
    - 个性化模式：自定义油门灵敏度、换挡节奏

## 四、380TSI专属4MOTION智能四驱系统
1. 硬件：博格华纳第五代Haldex电控多片离合器中央差速器，毫秒级扭矩分配
2. 常规铺装路面：前后轴95:5分配，以前驱行驶为主，兼顾油耗
3. 打滑/起步：最多50%扭矩传递至后轮，极端工况可分配90%动力至后轴
4. 轮间辅助EDS电子差速锁：单侧车轮空转时制动打滑车轮，扭矩集中至有附着力车轮，具备轻度交叉轴脱困能力
5. 专属驾驶模式：雪地、越野模式，调整油门响应、四驱介入优先级

## 六、动力安全防护机制
1. EA211/EA888发动机多级智能温控，高温爬坡、长时间拥堵分级限制输出功率；熄火后涡轮水冷延时散热，延长涡轮寿命
2. DQ381变速箱实时监测离合器油温，重度拥堵自动限制扭矩输出，避免离合器过热磨损
3. ESC车身稳定系统实时修正车身姿态，抑制起步打滑、弯道推头、高速横风偏移
4. 多重电控容错保护：发动机爆震保护、涡轮超压保护、冷却系统故障限功率输出
