# 极狐贝塔S3（极狐S3）动力参数介绍
## 一、动力架构总览
全系**400V纯电前置前驱**，统一前置永磁同步油冷单电机，提供**充电买断电池、换电租电**两套补能方案；标配全域热泵温控、多级动能回收、2.2kW V2L对外放电；底盘前麦弗逊独立悬架+后扭力梁非独立悬架。

## 二、全系统一电机性能参数
- 电机类型：前置扁线永磁同步油冷电机
- 峰值功率：159kW（216Ps）
- 峰值扭矩：250N·m
- 减速机构：电动车单速固定齿比减速器
- 官方0-100km/h加速：7.1s（区间6.9-7.6s，随配置/轮胎小幅浮动）
- 最高电子限速：165km/h
- 官方CLTC综合百公里电耗：低至11kWh

## 四、换电版（宁德时代巧克力可换电池）
- 电池规格：54kWh磷酸铁锂巧克力电池
- CLTC纯电续航：535km
- 补能方式：全自动99秒极速换电，免下车免插枪；仅支持家用交流慢充，无直流快充接口
- 购车方案：整车买断 / 车电分离电池月租，电池衰减、维护由换电站统一负责

## 五、电池全域热管理与安全
1. 全天候集成液冷热泵系统，-30℃低温自动预热、高温主动散热，电芯温差控制±2℃，降低冬季续航衰减；
2. 磷酸铁锂极光电池，多层气凝胶隔热防护，IP68级防尘防水，通过针刺、挤压、火烧极限安全测试；
3. 24小时实时监测电芯电压、温度、压差，碰撞瞬间切断高压回路，杜绝漏电风险；
4. 三电质保：充电版整车三电8年/15万公里；换电版电池统一场站维保，无需个人承担衰减损耗。

## 六、动能回收与电控制动系统
1. 多级可调能量回收强度：节能模式回收力度最强，支持城市单踏板行驶；运动模式回收减弱，滑行距离更长；
2. CRBS机电协同制动，日常减速90%依靠电机反向发电储能，减少刹车片磨损；
3. 长下坡自动提升回收功率，稳定车速同步补电；制动末端舒适刹停调校，抑制点头、晕车。

## 七、驾驶模式
6种可切换动力模式：舒适、极致节能、运动、湿滑路面、自适应、自定义，匹配不同路况输出逻辑。

## 八、动力安全防护机制
1. 前置油冷电机独立循环散热，城市通勤、高速持续巡航无热衰减；
2. 400V高压线束分层绝缘密封，高压回路故障毫秒级断电保护；
3. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；
4. 标配爆胎稳定控制系统，高速爆胎抑制车身跑偏失控。
