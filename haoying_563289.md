## 一、三套动力分项硬件参数
### （一）240TURBO 1.5T VTEC TURBO燃油版（L15CK）
1. 发动机基础参数
- 排量：1498mL 直列四缸涡轮增压，VTEC可变气门、水冷涡轮
- 发动机最大功率：142kW（193Ps）/6000rpm
- 发动机峰值扭矩：243N·m /1800–5000rpm
2. 传动系统：CVT无级变速箱（模拟7速）
3. 驱动形式：前置前驱，可选适时四驱
4. 性能与能耗
- 0‑100km/h加速：约8.9s
- 最高车速：198km/h
- WLTC综合油耗：两驱7.41L；四驱7.71L
- 油箱容积：53L

### （二）第四代i-MMD e:HEV 2.0L油电混动（LFB21阿特金森）
1. 混动专用2.0L自吸发动机
- 发动机最大功率：110kW（150Ps）/6100rpm
- 发动机峰值扭矩：183N·m /4500rpm，压缩比13.9:1，高热效率
2. 电驱单元
- 前置永磁同步驱动电机：峰值135kW，峰值扭矩335N·m
- 发电电机峰值120kW，两挡E‑CVT混动专用变速箱
3. 系统综合最大功率：135kW
4. 驱动区分：前驱 / AWD电子四驱（后桥独立电机）
5. 性能与能耗
- 0‑100km/h加速：两驱8.3s；四驱8.6s
- WLTC综合油耗：两驱5.49L；四驱5.85L
- 油箱容积：53L，无外接充电需求

### （三）SPORT HYBRID e+ 插电混动（e:PHEV，四驱标配）
1. 混动发动机同e:HEV 2.0L阿特金森（110kW / 183N·m）
2. 前后双电机四驱布局
- 前驱动电机135kW / 335N·m；后桥辅助电机
- 系统综合最大功率158kW，综合峰值扭矩335N·m
3. 变速箱：两挡E‑CVT混动专用
4. 动力电池：大容量三元锂液冷电池
- CLTC纯电续航85km，支持家用交流、公共直流快充
5. 性能与能耗
- 0‑100km/h加速：7.9s
- WLTC综合油耗：1.3L，WLTC馈电油耗5.6L
- 油箱容积：48L，标配3.3kW V2L对外放电

## 二、三套动力工作逻辑
1. **1.5T燃油版**
全程发动机驱动，CVT线性平顺；涡轮1800rpm即可输出满扭矩，城市超车轻快；适时四驱依靠多片离合分配动力，雨雪、轻度烂路提升抓地力，适合年均里程较少用户。
2. **第四代i-MMD油电混动（以电驱为主）**
- 低速拥堵：电机单独驱动，发动机仅发电，零油耗、静谧平顺；
- 中高速巡航：发动机高效区间直驱车轮，富余电能回充电池；
- 急加速满载：发动机+电机并联满功率输出；
- 减速刹车：电机反向发电储能，降低刹车片磨损；
- AWD四驱版本后桥独立电机，起步、打滑自动分配后轮动力。
3. **e:PHEV插电混动**
- 日常通勤纯电四驱行驶，完全不用烧油；
- 高速长途发动机直驱，亏电动力无明显衰减；
- 户外露营支持大功率外放电，兼顾家用与户外出行。
