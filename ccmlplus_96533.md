# 2026款哈弗猛龙PLUS动力参数介绍（第二代Hi4插混轻越野SUV，全系双电机电四驱）
## 一、动力总成整体概述
哈弗猛龙PLUS搭载**第二代Hi4三擎九模智能电混架构**，全系标配1.5T米勒循环混动专用发动机+前后双永磁同步电机+2挡DHT混动变速箱，无两驱版本；提供23.7kWh、33.7kWh磷酸铁锂、44.7kWh三元锂三档电池，加注92#无铅汽油，支持直流快充、6kW V2L户外外放电。整车质保3年/10万公里，三电系统8年/15万公里，兼顾城市纯电通勤、家用满载、轻度越野露营场景。

### 1、1.5T混动专用发动机（全系统一）
- 型号：GW4B15M 1.5T四缸涡轮增压，米勒循环+低压EGR废气再循环
- 最大功率：115kW（156Ps）/5500rpm，峰值扭矩243N·m/1800-4000rpm
- 变速箱：2挡DHT混动专用变速箱，低速纯电、高速发动机直驱双档位，传动效率更高
- 油箱容积55L，WLTC亏电油耗6.2L/100km

### 2、前后双电机电驱系统（全系标配四驱）
- 前永磁同步驱动电机：70kW / 160N·m，集成于2挡DHT前桥
- 后桥独立P4永磁同步电机：150kW / 350N·m，独立驱动后轮
- 系统综合最大功率：330kW（449Ps），综合峰值扭矩750N·m
- 性能：顶配255版0‑100km/h加速5.8s，中配195版6.0s，入门140版6.2s；最高车速190km/h电子限速

### 3、三档电池与续航版本

## 二、第二代Hi4电四驱工作逻辑（三擎九模）
1. **纯电两驱/四驱**：电量充足时发动机不启动，单前电机两驱省电，湿滑路面自动切换双电机四驱；
2. **串联发电模式**：低速拥堵、低电量，1.5T发动机稳定高效区间发电，电机驱动车轮；
3. **并联全力输出**：急加速、爬坡、脱困，发动机+前后双电机同步释放全部扭矩；
4. **高速直驱模式**：60km/h以上切换2挡发动机直驱，减少能量转换损耗，降低馈电油耗；
5. **四驱扭矩矢量分配**：0-100%前后轴毫秒级动态分配，搭配后桥牙嵌锁，单侧车轮打滑可锁止后桥，实现交叉轴脱困；
6. **能量回收**：滑行、刹车电机反向发电，回收动能存入电池，减少刹车片磨损。

1. 悬架结构：全系**前麦弗逊独立悬架+后多连杆独立悬架**；Ultra顶配搭载电控阻尼减振+路面预瞄系统，可实时调节底盘软硬，兼顾城市舒适与烂路支撑。
2. 核心越野通过参数：
    - 空载最小离地间隙221mm，最大涉水深度580mm；
    - 接近角25°、离去角32°，最小转弯半径5.7m；

3. 转向制动：EPS电动随速助力转向；前通风盘+后实心刹车盘，搭载机电协同动能回收制动；标配ABS、ESP车身稳定、上坡辅助、数字胎压显示。

## 五、三电安全防护说明
1. 电池布置于底盘中部，底部配备高强度钢制防护底板，抵御托底磕碰；电池包IP67防尘防水，支持短途涉水。
2. 全域液冷温控系统，快充主动降温、冬季低温自动预热，缩小四季续航衰减；顶配三元锂版本配备热泵空调，冬季续航提升明显。
3. 24小时BMS电池管理系统实时监控电芯电压、温度、压差；车辆碰撞瞬间切断高压回路，具备过压、欠压、过流、过热多重电控保护。
