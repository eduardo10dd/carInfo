# 2026款广汽丰田威兰达 动力参数介绍
基于TNGA-K平台，提供**2.0L燃油、2.0L第五代THS双擎、2.5L第五代THS双擎（E-FOUR四驱）**三套动力，全系加注92#无铅汽油；整车3年/10万公里质保，混动动力电池8年/20万公里质保，分为前驱、四驱两种驱动形式。

## 一、三套动力分项硬件参数
### （一）2.0L Dynamic Force 燃油版（M20C）
1. 发动机基础参数
- 布局：直列四缸自吸，D-4S双喷射、VVT-iE电动气门，压缩比13:1，热效率40%
- 最大功率：126kW（171Ps）/6600rpm
- 峰值扭矩：206N·m /4400–4900rpm
2. 传动系统：Direct Shift-CVT模拟10速无级变速箱
3. 驱动区分
- 两驱：WLTC综合油耗6.29L/100km，零百加速约9.9s
- 适时四驱：WLTC综合油耗6.89L/100km
4. 油箱容积：55L，最高车速180km/h

### （二）2.0L 第五代THS II 双擎（前驱/可选E-Four四驱）
1. 混动专用2.0L阿特金森发动机（M20F）
- 发动机最大功率：112kW（152Ps）/6000rpm
- 发动机峰值扭矩：188N·m /4400–5200rpm
2. 前置永磁同步驱动电机
- 电机峰值功率：83kW，峰值扭矩206N·m
3. 整套系统综合数据
- 系统综合最大功率：144kW（196Ps）
- 变速箱：E-CVT行星齿轮电子无级混动变速箱
4. 动力电池：小型风冷镍氢电池
5. 性能与能耗
- 两驱WLTC油耗4.59L/100km，零百加速约8.5s
- E-Four四驱WLTC油耗4.74L/100km
- 油箱55L，满油综合续航可达1500km

### （三）2.5L 第五代THS II 双擎（全系标配E-FOUR电子四驱）
1. 混动专用2.5L阿特金森发动机（A25D）
- 发动机最大功率：136kW（185Ps）/6000rpm
- 发动机峰值扭矩：236N·m /4200–4700rpm
2. 双电机布局（前+后桥独立电机）
- 前电机：83kW / 206N·m；后桥驱动电机：40kW
- 系统综合最大功率：174kW（237Ps），电机综合总扭矩391N·m
3. 变速箱：E-CVT电子无级混动变速箱
4. 性能与能耗
- 0‑100km/h实测加速8.3s，WLTC综合油耗4.91L
- 四驱逻辑：前后扭矩100:0~0:100毫秒级智能分配，带TRAIL越野脱困模式
- 油箱容积55L，最高车速180km/h

## 二、三套动力工作逻辑
1. **2.0L燃油版**
全程发动机直驱，CVT线性平顺，无换挡顿挫；适时四驱版本多片离合中央差速器，雨雪、轻度烂路提升抓地力，适合年均里程少、追求低维保成本用户。
2. **2.0L THS双擎（家用主力）**
- 低速拥堵：纯电机驱动，发动机不启动，零油耗、静谧平顺；
- 中高速巡航：发动机维持高效区间直驱，富余电能回充电池；
- 急加速满载：发动机+电机协同输出，动力无衰减；
- 减速刹车：电机反向发电储能，减少刹车片磨损。
3. **2.5L E-FOUR四驱双擎（顶配高性能）**
基础混动逻辑同2.0L双擎，后桥独立电机单独驱动后轮；起步、打滑、爬坡自动向后轴分配动力，抑制推头、打滑；2.5L发动机高速再加速储备更强，适合北方雨雪、经常满载长途用户。

## 四、全系驾驶模式
1. 燃油版：经济、标准、运动；四驱额外雪地模式；
2. 双擎混动版：经济、标准、运动、EV纯电模式；
全系两档动能回收强度可调，混动车型回收强度对油耗影响更明显；四驱车型联动四驱扭矩分配逻辑，雪地模式降低动力输出防止打滑。
