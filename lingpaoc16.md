# 2027款零跑C16 动力参数介绍（增程版+纯电版双架构）
## 一、动力总览
零跑C16提供**增程式混动（后置后驱）、800V纯电（后置后驱）**两套动力；增程搭载1.5L四缸增程器仅发电、不直驱车轮，全程电机驱动；全系标配液冷电池、3.3kW V2L对外放电、多级动能回收；增程加注92#汽油，纯电全系800V碳化硅高压平台。

# 第一部分：增程版（主流家用，分200km/280km纯电续航）
## 1. 全系通用1.5L自吸增程器（H15R）
- 排量：1498mL 直列四缸自然吸气
- 增程器最大功率：70kW（95Ps）
- 热效率：40%，仅负责发电，不驱动车轮
- 燃油标号：92#无铅汽油
- 优化NVH调校，增程器介入震动、噪音抑制优秀

## 2. 驱动电机参数
### 老款200续航版（28.4kWh电池）
- 后置永磁同步电机：峰值170kW / 320N·m
- 0-100km/h加速：8.5s
- CLTC纯电续航：200km，综合续航1095km
- WLTC馈电油耗：6.5L/100km
- 快充30%-80%：30分钟

### 新款280长续航版（38.9/38.7kWh磷酸铁锂，400V快充平台）
- 后置永磁同步油冷电机：峰值200kW（272Ps）/ 360N·m
- 0-100km/h官方加速：8.3s
- CLTC纯电续航：280km，CLTC满油满电综合续航1230km
- WLTC馈电综合油耗：6.29L/100km
- 直流快充峰值120kW，30%-80%仅19分钟
- 油箱容积65L

## 3. 增程四大工作模式
1. **EV纯电模式**：电量充足增程器停机，电机单独驱动，市区零油耗；
2. **串联增程模式**：低电量增程器稳定高效区间发电，电能供给电机；
3. **全力输出模式**：急加速/爬坡，电池+增程器同步供电，释放满功率；
4. **滑行/制动回收**：松油门、刹车电机反向发电储能。

# 第二部分：纯电版（800V碳化硅高压平台，后置单电机）
## 1. 电机统一参数
- 电机布局：后置扁线永磁同步油冷电机
- 峰值功率：230kW（313Ps）
- 峰值扭矩：360N·m
- 0-100km/h官方加速：5.9s
- 最高电子限速：180km/h
- 单速固定齿比减速器，动力全程线性无顿挫

## 2. 电池与补能参数
- 电池容量：81.9kWh磷酸铁锂
- CLTC纯电续航：630km
- 800V高压快充峰值250kW，30%-80%快充仅16-18分钟
- 6.6kW交流慢充，0-100%约11h
- 标配全域液冷温控，快充功率稳定不跳水

# 四、电池全域热管理与安全
1. 全系磷酸铁锂电池，液冷散热，IP68防尘防水，多层隔热防护，通过针刺、挤压、火烧极限测试；
2. 24小时实时监测电芯电压、温度、压差，碰撞瞬间切断高压回路；
3. 三电质保：非营运车辆三电系统终身质保（免责条款以官方为准）。

# 五、动能回收与电控制动系统
1. 多级可调能量回收：节能模式回收力度最强，支持城市单踏板行驶；运动模式回收减弱，滑行距离更长；
2. CRBS机电协同制动，90%日常减速依靠电机反向发电储能，减少刹车片磨损；
3. 长下坡自动提升回收功率，稳定车速同步补电；制动末端舒适刹停调校，抑制点头晕车。

# 六、动力安全防护机制
1. 后置油冷电机独立循环散热，满载高速、连续急加速无热衰减；
2. 增程版1.5L发动机多级温控，长时间馈电爬坡分级限功率，熄火延时水冷散热；
3. 400V/800V高压线束分层绝缘密封，故障毫秒级断电保护；
4. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；
5. 标配LMC爆胎稳定控制系统，高速爆胎抑制车身跑偏失控。
