# 2026款奇瑞风云T9L动力参数介绍（鲲鹏超能CDM 6.0插混中型SUV，前驱/四驱双版本）
## 一、动力总成整体概述
风云T9L全系搭载**鲲鹏超能CDM 6.0超级混动系统**，统一配备1.5T高热效率混动专用发动机+DHT260一挡混动变速箱，分为**前置前驱单电机**、**前后双电机智能四驱**两套动力；提供19.5kWh、32.7kWh两款液冷犀牛H磷酸铁锂动力电池，加注92#无铅汽油，支持800V高压快充、6kW V2L户外外放电，可上绿牌免购置税。整车质保3年/10万公里，首任车主动力电池终身质保，兼顾城市纯电通勤、满载家用、长途自驾、轻度越野场景。

### 1、1.5T混动专用发动机（全系统一）
- 型号：SQRH4J15 1.5T涡轮增压四缸
- 最大功率：115kW（156Ps），峰值扭矩220N·m
- 热效率：45.79%，14.5高压缩比，国Ⅵb排放
- 变速箱：DHT260一挡混动专用减速器，传动效率98.5%，油电切换0.1s无顿挫
- 油箱容积70L，支撑2000km超长综合续航

### 2、前置前驱单电机系统（135/230系列）
- 前驱动永磁同步电机：峰值功率260kW，峰值扭矩330N·m
- 系统综合最大功率375kW（510Ps），综合峰值扭矩550N·m
- 性能：官方0‑100km/h加速6.8~6.9s，最高车速190km/h
- 馈电油耗：CLTC工况3.9~4.4L/100km

### 3、前后双电机四驱性能版（200四驱超能版）
- 前电机：260kW / 330N·m；后P4驱动电机：175kW / 310N·m
- 系统综合最大功率550kW（748Ps），综合峰值扭矩860N·m
- 性能：实测零百加速4.83s，5秒级性能，高速再加速、爬坡脱困能力更强
- 馈电油耗：CLTC工况5.5L/100km

### 4、电池与续航分规格

#### 版本3：32.7kWh四驱专属电池

### 驾驶模式
纯电EV、智能混动、强制保电、运动、雪地、自定义；两档动能回收强度可调，强回收支持单踏板滑行。

## 二、CDM 6.0混动系统工作逻辑
1. **纯电通勤模式**：电量充足发动机不启动，电机单独驱动，市区零油耗、静谧平顺；
2. **串联发电模式**：低速拥堵、低电量工况，1.5T发动机稳定高效区间发电，电能供给电机驱动车轮；
3. **并联全力输出**：急加速、满载爬坡，发动机+电机同步释放全部动力；
4. **高速直驱模式**：匀速高速巡航，发动机直驱车轮，减少能量转换损耗，降低馈电油耗；
5. **四驱扭矩分配**：四驱版毫秒级动态分配前后轴扭矩，雨雪、泥泞路面抑制打滑，EDS电子限滑提升交叉轴脱困；
6. **能量回收**：滑行、刹车电机反向发电，回收动能存入电池，减少刹车片磨损。

## 四、底盘与制动（动力配套）
1. 悬架结构：全系**前麦弗逊独立悬架+后五连杆独立悬架**，家用舒适调校，兼顾高速侧向支撑；四驱版优化后悬架衬套，烂路滤震更好。
2. 通过性参数：轴距2920mm，最小转弯半径5.7m，标配540°全景透明底盘、蠕行脱困模式。
3. 转向制动：EPS电动随速助力转向；前通风盘+后实心刹车盘，搭载CRBS机电协同动能回收制动；标配ABS、ESP车身稳定、上坡辅助、数字胎压显示。

## 五、三电安全防护说明
1. 犀牛H磷酸铁锂电池布置于底盘中部，底部配备高强度钢制防护底板，抵御托底磕碰；电池包IP68防尘防水，支持短途涉水。
2. 全域液冷直冷直热温控系统，快充主动降温、冬季低温自动预热，大幅缩小四季续航衰减。
3. 24小时BMS电池管理系统实时监控电芯电压、温度、压差；车辆碰撞瞬间切断高压回路，具备过压、欠压、过流、过热多重电控保护。
4. 水冷集成式高压电驱总成，长时间满载高速、连续爬坡工况动力输出稳定无衰减。
