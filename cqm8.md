# 2026款广汽传祺M8 动力参数介绍
## 一、动力总成分类
传祺M8提供**2.0T燃油、2.0TM油电混动、2.0T插电混动（GMC400两挡DHT）**三套动力，全系加注92#无铅汽油，前置前驱布局；整车3年/10万公里质保，插混三电系统8年/16万公里质保，插混标配6kW V2L对外放电。

### （一）400T 2.0T 纯燃油版（领秀/大师燃油）
1. 发动机基础参数
- 型号：钜浪2.0TGDI涡轮增压直列四缸
- 最大功率：185kW（252Ps）/5250rpm
- 峰值扭矩：400N·m /1750–4000rpm
- 核心技术：350bar高压直喷、双流道涡轮、双平衡轴、GCCS燃烧控制专利
2. 传动系统：爱信8速手自一体变速箱（8AT）
3. 性能与能耗
- 0‑100km/h加速：约9.5s
- 最高车速：200km/h
- WLTC综合油耗：领秀8.85L/100km；大师8.95L/100km
- 油箱容积：65L

### （二）2.0TM HEV油电混动（大师混动）
1. 混动专用发动机
- 型号：2.0TM米勒循环混动专用机
- 发动机最大功率：140kW（190Ps）
- 发动机峰值扭矩：330N·m
- 热效率40.23%，水冷EGR、低压废气再循环
2. 驱动电机：前置永磁同步电机134kW / 300N·m
3. 系统综合最大功率：274kW，综合峰值扭矩630N·m
4. 变速箱：E‑CVT电子无级混动变速箱
5. 动力电池：小型风冷镍氢电池
6. 性能与能耗
- 0‑100km/h加速：8.8s
- WLTC亏电综合油耗：6.15L/100km，满油续航1057km
- 油箱容积：65L

### （三）2.0T PHEV插电混动（宗师插混，GMC400两挡DHT）
分**短续航136km、长续航248km**两款电池版本，统一2挡混动专用变速箱。
1. 混动发动机（同HEV 2.0TM）
- 发动机最大功率140kW，峰值扭矩330N·m
2. 前驱动电机分高低功率
- 标准版：134kW / 300N·m，系统综合274kW / 630N·m，零百8.8s
- 激光雷达高配：160kW / 315N·m，系统综合300kW / 645N·m，零百8.0s
3. 动力电池参数
- 短续航版：25.57kWh三元锂，CLTC纯电136km，30%-80%快充30分钟
- 长续航版：38.88kWh三元锂，CLTC纯电248km，30%-80%快充8分钟（高压快充）
4. 能耗与续航
- WLTC亏电油耗：6.05–6.38L/100km
- 满油满电综合续航：1160–1177km
- 油箱容积：50L
- 标配6kW大功率V2L外放电

## 二、三套动力工作逻辑
1. **2.0T燃油版**：全程发动机驱动，爱信8AT宽齿比，高速巡航转速低、静谧性好，适合年均里程少、长途高速居多用户。
2. **2.0TM HEV油电混动**
    - 低速拥堵：电机单独驱动，发动机不启动，零油耗、无顿挫；
    - 中高速巡航：发动机高效区间直驱，富余电能回充电池；
    - 急加速满载：油电协同输出，满载7人动力无衰减；
    - 减速刹车：电机反向发电储能，减少刹车片磨损。
3. **GMC400插混两挡DHT**
    - 城市通勤纯电行驶，日常代步不用烧油；
    - 中低速一挡电驱为主，高速切换二挡发动机直驱，降低能量损耗；
    - 亏电状态动力、加速性能与满电差距极小，无混动乏力通病；
    - 支持6kW户外放电，露营可带动全套家电。
