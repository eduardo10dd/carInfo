# 2026款凯迪拉克CT5 动力参数介绍（Markdown）
## 一、整车基础概况
凯迪拉克CT5基于**通用Alpha后驱运动平台**打造，定位豪华中型运动后驱轿车，4门5座布局，车身尺寸4930×1883×1453mm，轴距2947mm；全系统一搭载2.0T可变缸涡轮增压动力+纵置10AT手自一体变速箱，**前置后驱**布局，无四驱版本；统一加注95#高标号汽油；分城市风尚、城市运动、赛道性能三大配置梯度；底盘全系前双球节麦弗逊+后五连杆铝合金独立悬架，高配赛道版搭载MRC第四代磁流变电磁悬架、mLSD机械式限滑差速器、Brembo四活塞高性能制动；主打50:50前后轴荷比、后驱操控、美系运动豪华，整车3年/10万公里质保。

## 二、2.0T LSY可变缸涡轮增压发动机（全系通用）
### 发动机硬件参数
- 型号：LSY 2.0L直列四缸双涡管涡轮增压
- 最大功率：174kW（237Ps）/5000rpm
- 峰值扭矩：350N·m /1500–4000rpm，低转速即可释放满扭矩，城市起步、高速再加速储备充足
- 核心技术：Tripower三段式可变气门管理，智能切换**四缸高性能、四缸经济、两缸超经济**三种工况；35MPa缸内直喷、水冷涡轮、冷却EGR、全铝轻量化缸体，兼顾动力与燃油经济性
- 油箱容积：66L，长途续航充足
- 匹配传动：纵置10速手自一体AT变速箱，齿比绵密，支持手动换挡拨片，低速平顺、高速巡航转速更低，传动效率稳定，无双离合过热隐患

### 整车性能与能耗
- 官方0-100km/h加速：7.3s
- 最高电子限速：240km/h
- WLTC综合油耗：7.51–7.78L/100km
- 四种驾驶模式：舒适/运动/雪地/自定义，可调节动力输出、变速箱换挡逻辑、转向阻尼

## 三、全系配置动力硬件差异（低配/中配/赛道顶配）
### 1. 城市风尚/城市运动版（低配、中配）
1. 悬架：普通液压减振器，无电磁悬架
2. 差速器：开放式普通后桥差速器，无机械限滑
3. 制动：前后FNC通风盘、双活塞普通制动卡钳
4. 配重：标准50:50前后轴荷比，铝合金前后悬架控制臂

### 2. 赛道性能版（顶配运动强化）
1. **MRC第四代磁流变电磁悬架**：每秒1000次路面扫描，毫秒级调节减振阻尼，弯道大幅抑制侧倾，兼顾家用舒适与极限操控
2. **mLSD机械式后桥限滑差速器**：左右后轮扭矩实时分配，过弯、湿滑路面减少内侧车轮打滑，提升出弯加速能力
3. **Brembo四活塞前制动卡钳**：340mm大尺寸打孔通风盘，抗热衰减更强，百公里制动距离低至33.5米
4. 专属运动底盘调校、运动化转向阻尼、高性能运动轮胎

## 五、后驱操控底盘核心系统
1. **悬架结构**：前双球节麦弗逊独立悬架（双支点优化转向几何），后五连杆独立悬架，转向节、上下控制臂大量铝合金材质，降低簧下质量，提升悬架响应速度；50:50黄金轴荷比，转向中性，减少推头/甩尾问题
2. **转向系统**：EPS电动助力转向，运动模式加重转向阻尼，指向精准；小转弯半径，山路连续弯道操控灵活
3. **车身稳定系统**：StabiliTrak动态车身稳定、TCS牵引力控制、坡道起步辅助、自动驻车；雪地模式限制动力输出，优化低附着力路面行驶稳定性
4. **制动系统**：全系FNC防锈通风刹车盘，顶配升级Brembo竞技级制动，连续激烈驾驶无明显热衰减

## 六、发动机动力安全防护机制
1. LSY发动机独立水冷循环，涡轮配备延时冷却回路，长时间山路爬坡、激烈驾驶熄火后持续散热，降低涡轮热衰减；
2. 多重电控分级保护：爆震、水温过高、机油压力异常自动限制发动机输出功率，保护硬件；
3. 10AT变速箱内置独立油冷散热，长时间低速蠕行、山路连续换挡不易过热，变速箱寿命稳定性更强；
4. 后驱专属ESC标定，弯道、雨雪路面实时控制后轮动力输出，避免后驱打滑失控，兼顾运动乐趣与家用安全。
