# 2026款皇冠陆放 动力参数介绍
基于TNGA-K平台，提供**2.5L第五代THS II双擎混动（前驱/E-Four电子四驱）、2.0T涡轮增压燃油（前驱/DTV动态矢量四驱）**两套动力；混动加注92#汽油，2.0T燃油建议加注95#汽油；整车3年/10万公里质保，混动镍氢动力电池8年/20万公里质保。

## 一、两套动力分项硬件参数
### （一）2.5L 第五代THS II 智能电混双擎（A25H阿特金森）
1. 混动专用2.5L自吸发动机
- 排量：2487mL直列四缸阿特金森循环，压缩比14:1，最高热效率41%
- 发动机最大功率：139kW（189Ps）/6000rpm
- 发动机峰值扭矩：236N·m /4200–4700rpm
2. 电驱单元区分
- **两驱版**：前置单永磁同步电机，峰值134kW / 270N·m
- **E-Four四驱版**：前电机134kW，后桥独立辅助电机40kW，前后扭矩100:0~20:80智能分配
3. 整套系统综合数据
- 系统综合最大功率：181kW（246Ps）
- 变速箱：行星齿轮E‑CVT电子无级混动变速箱
4. 动力电池：风冷镍氢电池，无外接充电需求
5. 性能与能耗续航
- 0‑100km/h加速：两驱8.1s；四驱7.6s
- WLTC综合油耗：两驱5.82L/100km；四驱5.97L/100km
- 油箱容积：65L，满油综合续航超1000km
- 最高车速：180km/h

### （二）2.0T 涡轮增压燃油版
1. 发动机硬件
- 型号：直列四缸双流道涡轮，350Bar高压直喷
- 发动机最大功率：182kW（248Ps）/5000rpm
- 发动机峰值扭矩：380N·m /1800–4000rpm，1800rpm输出满扭矩
2. 传动：爱信8速手自一体8AT
3. 驱动区分
- 前驱：前置前驱
- DTV动态矢量四驱：可独立分配左右后轮扭矩，前后100:0~50:50大范围调节
4. 性能与能耗
- 0‑100km/h加速：两驱8.5s；四驱8.9s
- WLTC综合油耗：两驱7.81L；四驱8.75L
- 油箱容积：65L
- 最高车速：198km/h

## 二、两套动力工作逻辑
1. **2.5L THS双擎混动（以电驱为主）**
- 低速拥堵：电机单独驱动，发动机仅按需发电，零油耗、无顿挫；
- 中高速巡航：发动机维持高效区间直驱车轮，富余电能回充电池；
- 急加速/满载爬坡：发动机+电机并联满功率输出，7人满载动力无衰减；
- 减速/下坡：电机反向发电储能，降低刹车片磨损；
- E-Four四驱专属：起步、打滑、冰雪路面自动向后桥分配动力，抑制推头、提升行驶稳定。

2. **2.0T燃油版**
全程发动机驱动，爱信8AT液力变速箱换挡平顺、高速静谧；低转速高扭矩，满载、高速再加速储备充足；DTV矢量四驱兼顾城市行驶与轻度非铺装路面通过性，适合经常满载长途、追求更强动力用户。
