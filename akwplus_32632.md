# 2026款别克昂科威Plus动力参数介绍（中型五座SUV，2.0T可变缸+48V轻混，前驱/适时四驱）
## 一、动力总成整体概述
2026款昂科威Plus全系搭载**第八代Ecotec LSY 2.0T可变缸涡轮增压发动机**，标配48V MHEV轻混系统，匹配9速HYDRA-MATIC手自一体变速箱；分为**前置前驱**、**前置适时四驱**两套驱动版本，全系加注95#无铅汽油，油箱容积65L，整车质保3年/10万公里。主打家用满载通勤、高速长途、轻度非铺装路面出行，兼顾动力、静谧与燃油经济性。

### 1、2.0T发动机核心参数（全系统一）
- 型号：LSY 2.0L直列四缸涡轮增压，横置布局
- 最大功率：174kW（237Ps）/5000rpm
- 峰值扭矩：350N·m / 1500–4000rpm，宽泛扭矩区间，满载爬坡动力充足
- 核心可变缸技术：Tripower三段工况智能切换
  1. 四缸高性能：急加速、超车、爬坡全功率输出
  2. 四缸经济：城市常规行驶，平衡动力与油耗
  3. 两缸超经济：匀速巡航关闭两缸，降低长途油耗
- 48V轻混BSG一体机：负责平顺启停、起步扭矩补偿、滑行能量回收，削弱涡轮迟滞，优化低速平顺性

### 2、9AT变速箱亮点
9速HYDRA-MATIC智能变速箱，多挡位绵密齿比，高速巡航压低转速；支持手动换挡模式，换挡逻辑平顺，传动效率高，经过市场长期验证可靠性稳定。

### 3、分版本性能、能耗
#### （1）28T 两驱白金版（前置前驱）
- 官方0‑100km/h加速：8.1s
- 最高车速：210km/h
- WLTC综合油耗：7.88–7.99L/100km

#### （2）28T四驱白金 / 四驱艾维亚Avenir（适时四驱）
- 四驱结构：多片离合器式适时四驱，带后桥动力断开功能，日常两驱省油，湿滑/烂路自动分配四驱扭矩
- 官方0‑100km/h加速：8.1s
- 最高车速：210km/h
- WLTC综合油耗：8.27–8.37L/100km
- 艾维亚专属：FlexRide自适应底盘+CDC主动减振，多地形驾驶模式

### 驾驶模式区分
1. 两驱车型：舒适、运动、雪地三种模式；
2. 四驱艾维亚车型：舒适、运动、雪地、越野四种模式，变速箱与四驱扭矩同步联动调节。

## 二、2.0T+48V轻混+适时四驱工作逻辑
1. **城市低速通勤**：轻载自动切换两缸经济模式，48V电机实现无感启停，涡轮低转速快速介入，起步平顺无顿挫；
2. **匀速高速巡航**：四缸经济工况为主，9AT拉高挡位压低转速，滑行阶段电机反向发电储能，减少发动机负荷；
3. **急加速/满载超车**：瞬间切换四缸全功率输出，48V电机补充瞬时扭矩，350N·m扭矩持续输出，中后段加速储备充足；
4. **雨雪/轻度越野（四驱版）**：系统毫秒级接合多片离合器，前后轴动态分配扭矩，配合EDS电子限滑抑制车轮空转，提升湿滑路面稳定性；
5. **减速滑行**：松油门切断喷油，48V电机回收动能存入电池，减少刹车磨损，优化燃油消耗。

## 四、底盘与制动配套硬件
1. 悬架结构：全系**前麦弗逊独立悬架+后五连杆独立悬架**；前三角臂、转向节采用铝合金材质，降低簧下质量，提升悬架响应；艾维亚搭载CDC每秒100次阻尼实时调节，兼顾舒适与操控支撑。
2. 通过性参数：空载最小离地间隙188mm，最小转弯半径5.7m；轴距2833mm，大五座空间优势明显，标配全景透明底盘、陡坡缓降。
3. 转向制动：EPS电动随速助力转向；前通风盘+后通风盘刹车，eBoost电子助力制动；标配ABS、ESP车身稳定、上坡辅助、数字胎压显示。

## 五、48V轻混系统功能说明
1. 无感启停：等红灯、低速蠕行平稳熄火启动，消除传统燃油车启停抖动；
2. 扭矩补偿：起步、急加速电机瞬时输出辅助扭矩，缓解涡轮迟滞，低速动力响应更快；
3. 动能回收：滑行、制动回收动能存入48V电池，为车载电器、底盘电控供电，降低发动机负载；
4. 巡航减负：匀速行驶减少发动机喷油负荷，小幅降低综合工况油耗。
