# 2026款路虎卫士 动力技术参数介绍（燃油/轻混/PHEV插混/V8高性能全系列）
## 一、整车基础概况
全新路虎卫士基于**D7x全铝整合式承载车身**打造，分90短轴两门、110标准五门、130加长八座三种车身尺寸；全系前置全时四驱、采埃孚8AT自动变速箱、双速分动箱；提供**2.0T P300燃油、3.0T直六P400 48V轻混、P400e 2.0T插混、4.4T V8高性能**四大动力，柴油版本海外专供国内不引进；全系推荐95#无铅汽油；标配第二代全地形反馈系统、中央差速锁，高配搭载主动式电控后桥差速锁；底盘前双叉臂+后五连杆独立悬架，高配标配空气悬架，兼顾豪华城市通勤与重度越野能力。

## 二、全系动力总成详细参数
### 1. P300 2.0T 四缸涡轮增压（入门城市家用）
- 发动机型号：Ingenium 2.0T直列四缸涡轮增压
- 最大功率：221kW（300Ps）/5500rpm
- 峰值扭矩：400N·m /1500–4500rpm，低转扭矩充沛，城市代步平顺
- 变速箱：ZF 8HP 8速手自一体
- 驱动形式：全时四驱+双速分动箱（高低速扭矩放大）
- 0-100km/h加速：7.1s
- 最高时速：191km/h
- WLTC综合油耗：9.6–10.2L/100km
- 油箱容积90L，满油综合续航约880km
- 适配场景：城市通勤、轻度自驾，预算入门选择，无48V轻混结构

### 2. P400 3.0T 直列六缸+48V MHEV轻混（主力走量）
- 发动机型号：Ingenium 3.0L直六涡轮增压+48V轻混系统
- 最大功率：294kW（400Ps）/5500–6500rpm
- 峰值扭矩：550N·m /2000–5000rpm
- 48V轻混作用：起步助力、滑行能量回收、启停平顺，降低市区油耗
- 变速箱：ZF 8HP 8速手自一体
- 0-100km/h加速：5.9s
- 最高时速：191km/h
- WLTC综合油耗：9.2–9.8L/100km
- 标配空气悬架、主动后桥差速锁，越野硬件完整
- 适配场景：家用长途、中度越野、兼顾动力与豪华舒适

### 3. P400e 2.0T 插电混动PHEV（绿牌城市版）
#### 1）内燃机部分
2.0T涡轮增压四缸发动机，最大功率221kW（300Ps）
#### 2）后置驱动电机
电机峰值功率105kW，峰值扭矩260N·m
#### 整套混动综合性能
- 系统综合最大功率：297kW（404Ps）
- 系统综合峰值扭矩：640N·m
- 0-100km/h加速：6.1s
- 最高时速：191km/h
#### 电池、续航与补能
- 电池容量：19.2kWh三元锂电池
- CLTC纯电续航：110km，满足日常通勤零油耗
- WLTC综合油耗：2.8L/100km；亏电综合油耗8.2–9.0L/100km
- 油箱90L，满油满电综合续航超900km
- 充电规格：交流慢充0-100%约3.5h；无直流快充
- 工作模式：纯电EV、混动均衡、电量保持、越野保电四种模式，越野工况四驱机械结构不脱开

### 4. P635 4.4T V8双涡轮高性能旗舰
- 发动机：4.4L V8双涡轮增压
- 最大功率：467kW（635Ps）/6000rpm
- 峰值扭矩：750N·m /2500–5500rpm
- 变速箱：强化版ZF 8HP 8AT，承受超大扭矩
- 0-100km/h加速：5.4s
- 最高时速：240km/h
- WLTC综合油耗：12.5–13.2L/100km
- 专属升级：运动化底盘调校、高性能制动、专属动态模式、全地形增强程序
- 适配场景：性能玩家、长途高速、重度穿越，追求极致动力储备

## 四、核心四驱与越野传动系统
1. **全时四驱+双速分动箱**
    标配高低两档，低速挡扭矩放大2.93倍，岩石、陡坡攀爬大幅放大轮端扭矩；中央多片离合器式差速锁可100%锁止前后轴动力。
2. **主动式电控后桥差速锁（3.0T/PHEV/V8标配）**
    实时分配左右后轮扭矩，单侧车轮打滑瞬间锁止，交叉轴、泥地脱困能力大幅提升。
3. **第二代Terrain Response 2全地形反馈**
    自动/草地/泥泞/沙地/岩石/雪地/运动/节能8种预设模式+自定义全地形管家，自动匹配动力输出、悬架高度、制动逻辑、四驱锁止力度。
4. **ATPC全地形进程控制**
    越野蠕行，2–30km/h匀速稳定通过复杂路况，无需控制油门刹车。

## 五、底盘硬件与通过性参数
1. 悬架：前双叉臂独立悬架+后五连杆独立悬架；空气悬架升降行程145mm，标准离地间隙218mm，最高越野模式291mm。
2. 通过几何：接近角37.4°、离去角40°、通过角27.9°，最大涉水深度900mm，标配涉水感应预警。
3. 车身：D7x全铝整合车身，整车扭转刚度30000N·m/deg，远超传统非承载硬派SUV，兼顾轻量化与抗扭强度。
4. 制动：四轮通风盘制动，V8版本升级高性能大尺寸制动卡钳，长下坡、重度越野无热衰减。

## 六、插混P400e专属电池热管理与安全
1. 19.2kWh三元锂电池包布局底盘中部，配备独立液冷温控系统，低温预热提升冬季纯电续航；
2. 电池包IP68防尘防水，底部钢制防护底板，涉水、托底具备充足防护；
3. 24小时BMS电芯监测，碰撞瞬间切断高压回路，杜绝漏电风险；
4. 混动系统保留完整机械四驱传动，纯电、混动、燃油模式四驱结构全程联动，不丢失越野硬件能力。

## 七、动能回收与制动系统
1. **48V轻混/PHEV插混**：多级动能回收，滑行、制动回收电能储存至电池，降低市区油耗；越野模式回收力度减弱，保障脱困滑行自由度。
2. **燃油版**：无强电回收，制动踏板线性厚重，保留硬派越野传统制动质感。
3. 全系标配坡道缓降、坡道起步辅助、牵引稳定控制、侧倾稳定控制系统，山路、湿滑路面行驶更安全。

## 八、动力安全防护机制
1. Ingenium全系发动机水冷循环温控，长时间满载爬坡、高温越野分级限制输出功率，涡轮独立冷却回路，熄火延时散热；
2. ZF 8AT变速箱内置独立油冷散热，连续低速越野、长时间蠕行不易过热；
3. PHEV高压线束分层绝缘密封，故障毫秒级切断高压；多重电控保护：过压、欠压、超温自动限制电机与发动机动力；
4. 四驱系统与ESC车身稳定联动，碎石、雪地、泥泞路面实时分配四轮扭矩，抑制推头、甩尾失控。
