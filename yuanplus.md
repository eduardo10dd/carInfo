# 比亚迪元PLUS 动力参数介绍（2025老款前驱 + 2026新款800V后驱）
## 一、动力总览
分为两代车型：
1. **2025款 e平台3.0 前置前驱**：150kW单电机，400V平台，430/510km续航；
2. **2026新款 e平台3.0 EVO 后置后驱**：800V高压闪充，分200kW、240kW两种电机，540/630km续航；
全系磷酸铁锂刀片电池，标配2.2kW V2L对外放电、热泵空调；首任非营运三电系统终身质保。

# 三、全系刀片电池热管理与安全
1. CTB电池车身一体化磷酸铁锂刀片电池，IP68防尘防水，多层气凝胶隔热，通过针刺、挤压、火烧极限安全测试；
2. 全域宽温域热泵温控，-30℃低温自动预热、快充高负荷主动散热，降低冬季续航衰减；
3. 24小时实时监测电芯电压、温度、压差，碰撞瞬间切断高压回路，杜绝漏电风险。

# 四、动能回收与电控制动系统
1. 三级可调能量回收：标准/较强/单踏板，节能模式回收力度最大，城市代步可单踏板行驶；运动模式回收减弱，滑行距离更长；
2. CRBS机电协同制动，90%日常减速依靠电机反向发电储能，减少刹车片磨损；
3. 长下坡自动提升回收功率，稳定车速同步补电；制动末端舒适刹停调校，抑制点头晕车。

# 五、动力安全防护机制
1. 高功率电机搭载独立油冷循环散热，连续全油门、高速巡航无热衰减；
2. 400V/800V高压线束分层绝缘密封，故障毫秒级断电保护；
3. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；
4. 标配车身稳定控制系统，湿滑路面抑制车轮打滑、车身失控。
