# 腾势D9 动力参数介绍（DM-i插混 + EV纯电双架构）
## 一、动力总览
腾势D9分为**第五代DM-i插电混动**、**800V高压EV纯电**两大系列；插混提供前置前驱、智能电四驱，纯电同样单电机前驱/双电机四驱；全系标配6kW V2L对外放电、云辇-C智能阻尼悬架、刀片电池全域液冷温控；插混加注92#汽油，首任非营运三电终身质保。

# 第一部分：DM-i 第五代超级插混（2026新款闪充版为主）
## 1. 全系通用1.5T混动专用发动机
- 型号：1.5T涡轮增压米勒循环
- 排量：1498mL，350MPa高压直喷、冷却EGR
- 发动机最大功率：115kW（156Ps）
- 峰值扭矩：245N·m
- 最高热效率45.3%，兼顾发电与高速直驱，匹配E-CVT电子无级变速箱

## 2. DM-i 两驱版（前置单电机）
- 前永磁同步电机：峰值200kW / 425N·m
- 系统综合最大功率：245kW
- 系统综合峰值扭矩：571N·m
- 官方0-100km/h加速：8.6s

- CLTC纯电续航：401km
- CLTC满油满电综合续航：1520km
- WLTC亏电油耗：6.35L/100km
- 快充：200kW直流，30%-80%仅25分钟

## 3. DM-i 四驱版（前200kW+后45kW双电机）
- 前电机：200kW / 425N·m；后电机：45kW / 110N·m
- 系统综合最大功率：299kW
- 系统综合峰值扭矩：681N·m
- 官方0-100km/h加速：7.3s
- 电池：40.06kWh刀片电池
- CLTC纯电续航：401km
- CLTC综合续航：1520km
- WLTC亏电油耗：6.65L/100km
- 补能规格、V2L功率与两驱一致

## 4. DM-i 四大工作模式
1. **EV纯电模式**：电量充足发动机停机，电机单独驱动，市区零油耗；
2. **串联增程模式**：低速低电量，发动机稳定高效区间发电供给电机；
3. **并联直驱模式**：高速巡航锁止E-CVT，发动机直驱车轮，降低电耗损耗；
4. **滑行/制动回收**：松油门、刹车电机反向发电，电能存入电池循环利用。

# 四、电池全域热管理与安全
1. CTB电池车身一体化磷酸铁锂刀片电池，IP68防尘防水，多层气凝胶隔热，通过针刺、挤压、火烧极限安全测试；
2. 800V全域直冷直热温控，-30℃低温自动预热、快充高负荷主动散热，缓解冬季续航衰减；
3. 24小时实时监测电芯电压、温度、压差，碰撞瞬间切断全部高压回路，杜绝漏电风险。

# 五、动能回收与电控制动系统
1. 三级可调能量回收：标准/较强/单踏板；节能模式回收力度最大，长下坡自动提升回收功率稳定车速；运动模式回收减弱，滑行距离更长；
2. CRBS机电协同制动，90%日常减速依靠电机反向发电储能，大幅减少刹车片磨损；
3. 坡道识别自动匹配回收力度，减轻长下坡刹车热衰减。

# 六、动力安全防护机制
1. 1.5T混动发动机多级智能温控，长时间馈电满载爬坡、高温拥堵分级限制输出功率；涡轮水冷熄火延时散热；
2. 前后电机独立油冷循环散热，连续全油门、高速长途无热衰减；800V碳化硅电控降低高压损耗；
3. 高压线束分层绝缘密封，高压回路故障毫秒级断电保护；
4. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；
5. ESC车身稳定系统联动四驱扭矩分配，抑制湿滑路面车轮打滑、车身失控。
