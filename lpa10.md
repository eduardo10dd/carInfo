# 零跑A10 动力系统完整参数
## 一、整车动力基础概况
- 车型定位：小型5门5座纯电家用SUV，零跑A纯电专属平台打造
- 驱动布局：全系前置永磁同步单电机前驱
- 电压平台：505km长续航版搭载800V高压快充架构；403km入门版400V平台
- 传动结构：电动车固定齿比单速减速器，高配七合一油冷集成电驱
- 质保政策：整车4年/12万公里；动力电池8年/15万公里
- 特色功能：多级可调动能回收、CTC2.0 Plus电池底盘一体化、全系标配高压直流快充
- 驾驶模式：经济/标准/运动三档动力模式，支持单踏板强回收
- 最高电子限速：160km/h，支持150km/h稳定高速巡航不衰减

## 二、两套电机动力性能参数
### 1. 403km入门版（舒享/悦享）
- 电机类型：前置永磁同步电机
- 峰值功率：70kW
- 峰值扭矩：150N·m
- 官方0-100km/h加速：12.2s
- 电池规格：39.8kWh磷酸铁锂电池，风冷散热
- CLTC纯电续航：403km
- 百公里综合电耗：11.3kWh
- 快充性能：30%-80%充电时长30分钟
- 慢充配置：6.6kW车载充电机，满充约6.5小时

### 2. 505km长续航版（悦享/激光雷达旗舰）
- 电机类型：前置七合一油冷永磁同步电驱
- 峰值功率：90kW
- 峰值扭矩：150N·m
- 官方0-100km/h加速：10.6s
- 电池规格：53kWh磷酸铁锂电池，液冷全域温控
- CLTC纯电续航：505km
- 百公里综合电耗：10.7kWh
- 快充性能：800V高压平台，峰值快充功率133kW，30%-80%仅16分钟，10分钟快充补能约150km
- 慢充配置：6.6kW车载充电机，满充约8.5小时

## 三、电池与热管理系统
1. 电池硬件
    - 电芯类型：磷酸铁锂，循环寿命长，成本更低，家用代步性价比突出
    - 散热区分：403km风冷散热；505km液冷液热一体化温控，高温稳定快充功率、低温自动预热电芯
    - CTC2.0 Plus一体化：电池包融合车身结构，提升整车刚性，降低车身重心
2. 充电特性
    - 长续航版支持2.5C高倍率快充，同价位少见800V高压平台
    - 家用交流慢充、公共直流快充双接口全系标配
    - 无V2L对外放电功能

## 四、电控与动能回收控制逻辑
1. 前驱动力特性
    - 电门调校线性柔和，城市0-60km/h起步响应轻快，适合拥堵通勤
    - 高速100km/h以上再加速动力储备偏弱，定位城市代步取向
2. 动能回收系统
    - 三档回收强度可调，强回收模式实现单踏板驾驶，减少刹车使用
    - 滑行、下坡、制动阶段自动回收电能，降低日常电耗
3. 动力保护机制
    - 油冷电驱散热效率优于传统水冷，长时间高速巡航不易动力限功率
    - 电池过压、欠压、过温多重电控保护；碰撞瞬间高压自动切断回路

## 五、底盘、悬挂与制动配套
1. 悬架结构（全系统一）
    - 前悬架：麦弗逊式独立悬架，带横向稳定杆
    - 后悬架：扭力梁式非独立悬架，家用舒适化调校，过滤路面细碎颠簸
2. 转向系统：电动助力转向，低速轻盈，小区、窄路掉头灵活
3. 制动系统
    - 前轮通风刹车盘、后轮实心刹车盘
    - 优先动能回收减速，高强度制动机械刹车介入
    - 标配ABS、ESP车身稳定、LMC高速爆胎稳定系统、坡道辅助、自动驻车
4. 底盘配套：数字实时胎压监测、底盘防腐防护涂层

## 六、三电安全防护
- 磷酸铁锂电芯，通过针刺、挤压、高温火烧安全测试
- 9横7纵4强环笼式车身，电池包底部加装高强度防护钢板
- 高压线束分层绝缘，整车实时绝缘监测，故障自动断电
- 低重心电池布局，降低高速过弯侧倾，提升行驶稳定性
