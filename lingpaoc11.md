# 2027款零跑C11 动力技术参数介绍（增程版+800V纯电双架构）
## 一、整车基础概况
零跑C11定位中型五座家用SUV，车身尺寸4780×1905×1675mm，轴距2930mm；全系**后置后驱**布局，分为**1.5L增程版、800V高压纯电版**两套动力；全系标配**前双叉臂+后五连杆独立悬架+FSD可变阻尼减振**、大陆Onebox线控制动；增程加注92#无铅汽油，全系支持3.3kW V2L对外放电；首任非营运车主三电系统终身质保，整车5年/15万公里质保；搭载自研800V碳化硅高压平台，CTC2.0 Plus电池一体化底盘。

## 第一部分：1.5L高效增程系统（300km纯电续航）
### 1. H15R 1.5L阿特金森增程发生器（全系增程通用）
- 排量：1498mL直列四缸自然吸气，全铝轻量化机身
- 增程器峰值发电功率：70kW（95Ps），仅负责发电，不直驱车轮
- 最高热效率40%，多重NVH减振降噪，增程介入噪音低至42dB
- 油箱容积47L大容量油箱，长途无续航焦虑

### 2. 后置永磁同步驱动电机
- 电机峰值功率：200kW（272Ps）
- 电机峰值扭矩：360N·m，起步瞬时满扭矩输出，城市超车轻快
- 冷却形式：360度智能分流油冷，连续满载行驶无热衰减
- 传动：单速固定齿比减速器，全程纯电行驶质感，无换挡顿挫
- 官方0-100km/h加速：7.6s
- 最高电子限速：170km/h

### 3. 电池、续航与补能规格
- 电池规格：41.7kWh磷酸铁锂电池，全域液冷热泵热管理
- CLTC纯电续航：300km，日常通勤可纯电零油耗出行
- CLTC满油满电综合续航：1220km
- WLTC亏电综合油耗：5.11L/100km
- 800V高压快充：30%-80%电量仅18分钟，峰值充电功率200kW
- 6.6kW交流慢充，0-100%满充约6.8h
- V2L外放电：3.3kW，支持露营电器、户外工具供电

### 增程四大工作模式
1. **EV纯电模式**：电量充足增程器停机，后置电机独立驱动，座舱静谧、零燃油消耗；
2. **串联发电模式**：低电量工况，1.5L增程器稳定高效区间发电，电能供给后驱电机；
3. **全力输出模式**：急加速、满载爬坡，电池大功率放电+增程器满负荷发电同步释放360N·m峰值扭矩，满/亏电动力差距极小；
4. **滑行/制动能量回收**：松油门、刹车电机反向发电，回收电能存入电池循环利用。

1. CTC2.0 Plus电池底盘一体化技术，取消传统模组上盖，车身扭转刚度提升15.5%，电池包防护更扎实；
2. 全域直冷直热一体化热泵系统，回收电机、电控余热加热电池，12分钟内将-20℃电池升温至最佳工作区间；
3. 24小时AI-BMS全域电芯监测，实时采集电压、温度、压差，碰撞2毫秒级切断全部高压回路；
4. 多层气凝胶隔热+2mm钢制底部护板，通过针刺、挤压、火烧极限安全测试，无热扩散风险。

## 五、动能回收与线控制动系统
1. 三级可调能量回收强度，城市代步强回收减少刹车使用，高速模式回收减弱，滑行距离更长；
2. 大陆Onebox IPB线控制动CRBS机电协同系统，90%日常减速依靠电机回收储能，大幅降低刹车片磨损；

1. 悬架：全系前双叉臂独立悬架（液压衬套）+后五连杆独立悬架，标配FSD可变阻尼减振，欧洲团队联合调校；过弯侧倾＜3°，兼顾舒适与操控；前四活塞制动卡钳，制动支撑性强；
2. 后置电机独立油冷散热，长时间高速、满载爬坡无热衰减；800V碳化硅电控降低高压电能损耗；
