# 2026款比亚迪汉 动力参数介绍（DM-i家用插混 / DM-p性能四驱 / EV纯电后驱/四驱）
## 一、整车基础概况
比亚迪汉定位中大型豪华新能源轿车，车身尺寸4995×1910×1495mm，轴距2920mm；分为**DM-i前驱省油插混、DM-p四驱性能插混、EV纯电后驱、EV纯电四驱高性能**四大动力系列。
- 插混车型搭载第五代DM混动系统，加注92#无铅汽油；
- EV纯电全系800V高压碳化硅平台，刀片磷酸铁锂液冷电池；
- 全系标配3.3kW V2L对外放电、全域热泵热管理；高配搭载云辇-C智能阻尼车身控制系统；
- 底盘统一前麦弗逊独立悬架+后五连杆铝合金独立悬架；首任非营运车辆三电系统终身质保。

## 二、汉DM-i 第五代DM-i前驱家用插混（主打低油耗长纯电）
### 1.5T骁云插混专用发动机（DM-i全系通用）
- 排量：1498mL直列四缸涡轮增压，米勒循环，最高热效率45.3%
- 发动机最大功率：115kW（156Ps）/5500rpm
- 发动机峰值扭矩：230N·m /1800–4000rpm
- 配套传动：EHS混动专用E-CVT无级变速箱，油电耦合无顿挫，支持高速发动机直驱降低能耗
- 油箱容积60L

### 前置水冷永磁同步驱动电机
- 电机峰值功率：160kW（218Ps），峰值扭矩310N·m
- 系统综合峰值功率：217kW
- 官方0-100km/h加速：6.7s
- 最高电子限速：180km/h

### 电池、续航与补能（两档续航）
1. **125km标准版**
    - 电池容量：18.3kWh磷酸铁锂刀片电池
    - CLTC纯电续航：125km；WLTC亏电油耗3.44L/100km
    - CLTC满油满电综合续航1350km
    - 直流快充峰值40kW，30%-80%约30分钟
2. **245km长续航版**
    - 电池容量：29.4kWh大容量刀片电池
    - CLTC纯电续航：245km，日常通勤可一周纯电零油耗
    - WLTC亏电油耗低至3.35L/100km，综合续航1560km
    - 直流快充峰值69kW，30%-80%仅25分钟
- 统一配置：6.6kW交流慢充、3.3kW V2L外放电

### DM-i五大工作模式
1. EV纯电模式：电量充足发动机停机，纯电行驶零油耗；
2. 串联增程模式：低速低电量，发动机稳定高效区间发电；
3. 高速直驱模式：巡航工况发动机直驱车轮，减少电能损耗；
4. 并联全力输出：急加速、满载爬坡，油电同步释放峰值动力；
5. 保电模式：强制发电储存电量，预留市区纯电出行；
6. 能量回收：三级可调回收强度，滑行、制动回收电能延长续航。

- 快充峰值69kW，30%-80%约25分钟，标配云辇-C减振、全套运动驾驶模式

## 六、云辇-C智能底盘与四驱系统
1. **悬架硬件**：全系前麦弗逊独立悬架+后五连杆铝合金独立悬架；云辇-C版本搭载连续可变阻尼减振，毫秒级调节悬架软硬，抑制刹车点头、过弯侧倾，兼顾舒适与操控。
2. **四驱逻辑（DM-p/EV四驱）**：前后电机独立扭矩分配，雨雪、湿滑路面毫秒级调节四轮动力，提升高速稳定性与起步抓地力。
3. **制动系统**：全系IPB线控制动CRBS机电协同，90%日常减速依靠电机回收储能；四驱高性能版升级大尺寸通风打孔刹车盘，长下坡依靠动能回收抑制刹车热衰减。
4. **驾驶模式**：经济/标准/运动/雪地/自定义，同步调节动力输出、悬架阻尼、四驱响应强度。

## 七、刀片电池全域液冷热管理与安全
1. 一体化直冷直热热泵热管理，回收电机、发动机余热预热电池，-10℃低温纯电续航保持率75%以上；EV 800V平台快充温控优化，大电流充电温升控制优秀。
2. 刀片电池IP68防尘防水，底部全覆盖钢制防护底板，通过针刺、挤压、浸水、火烧极限安全测试，无热蔓延风险。
3. 24小时AI-BMS全域电芯监测，实时采集电压、温度、压差，碰撞2毫秒级切断全部高压回路，杜绝漏电隐患。

## 八、三电系统安全防护机制
1. 全系水冷永磁电机+碳化硅电控总成，电驱综合效率最高97.5%，满载多人长途、激烈驾驶无热衰减；800V高压平台电能损耗相比400V降低30%。
2. DM系列1.5T混动发动机独立水冷循环，涡轮熄火延时冷却，高温长途爬坡自动限制输出保护硬件。
3. 高压线束分层绝缘密封，系统实时监测高压回路，故障毫秒级切断高压电源；多重电控分级保护：过压、欠压、过流、超温自动限制电机与发动机动力。
4. ESC车身稳定系统联动四驱扭矩分配，雨雪、碎石路面动态分配四轮扭矩，抑制推头、甩尾失控，提升豪华家用出行安全。
