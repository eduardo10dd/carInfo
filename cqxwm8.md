# 传祺向往M8 PHEV L 动力参数介绍
## 一、混动系统总览
全系搭载**星源GMC 3.0两档DHT插混系统**，分**1.5T两驱单电机、1.5T四驱双电机**两套动力，统一44.57kWh大容量三元锂弹匣电池；加注92#无铅汽油，国六b排放，最高电子限速190km/h。
驾驶模式：EV纯电/HEV混动/运动/电量保持；三档动能回收可调，支持单踏板模式；标配6kW V2L对外放电。

## 二、1.5T混动专用发动机（全系通用）
- 排量：1498mL 直列四缸涡轮增压
- 进气形式：双流道涡轮、350bar高压直喷、低压EGR、阿特金森循环
- 最大功率：125kW（170Ps）/5500rpm
- 峰值扭矩：245N·m /2500–4000rpm
- 极限热效率：47.26%，独立水冷中冷+智能热管理，馈电长途动力无衰减

## 三、GMC 3.0 两档DHT混动变速箱核心
1. 双档位平行轴串并联结构，支持纯电、串联增程、1/2档发动机直驱、并联动力、驻车发电多模式切换
2. 低速电机驱动平顺静谧，高速2档直驱降低能量损耗，亏电油耗优于单档混动
3. 独立油冷散热，满载7人长途、连续爬坡不易过热；升降挡无顿挫，兼顾商务舒适与动力响应

## 四、两套驱动版本完整动力参数

- 系统综合最大功率：410kW（557Ps），综合峰值扭矩770N·m
- 官方0-100km/h加速：6.17s
- CLTC纯电续航：235km；WLTC纯电续航191km
- WLTC亏电百公里油耗：6.35L
- CLTC满油满电综合续航：1200km+
- 四驱优势：湿滑、雨雪路面实时分配四轮扭矩，满载爬坡、高速超车动力储备更强

## 五、44.57kWh弹匣三元锂电池与补能参数
1. 电池规格：宁德时代骁遥三元锂，3C快充倍率，弹匣式全域防护，CTP直冷一体化热管理
2. 快充性能：60kW高压直流快充，SOC 30%-80%仅15分钟，15分钟补能约140km续航
3. 慢充：6.6kW交流车载充电机，完整充满时长6.5h，支持低谷预约充电
4. 外放电：全系6kW VTOL移动电站，支持露营电磁炉、投影仪、户外工具供电，过载自动断电保护
5. 安全热管理：全域液冷液热宽温域热泵，-30℃低温脉冲预热、高温主动散热；碰撞2ms切断高压回路，24小时实时监测电芯压差、绝缘状态，通过针刺、挤压、火烧极限安全测试

## 六、动能回收与电控制动系统
1. 三档可调动能回收，强回收模式支持城市单踏板行驶，大幅减少刹车片磨损
2. CRBS电制动+机械制动协同，90%日常减速依靠电机回收电能，降低长期用车损耗
3. 长下坡自动提升回收功率，稳定车速同时储存电能，减轻刹车负荷
4. 路况预判能量管理：识别拥堵、长上坡、高速路段，动态调节油电分配逻辑，提升续航达成率

## 七、动力安全防护机制
1. 1.5T发动机多级温控保护，长时间满载、高温爬坡分级限功率，电子废气旁通阀精准控制涡轮压力
2. GMC3.0两档DHT独立油冷散热回路，满载7人长途、激烈加速无热衰减
3. 前后双电机独立油冷散热，四驱持续高负荷输出性能稳定
4. 高压线束分层绝缘密封，故障瞬时切断高压回路，杜绝漏电风险
