# 长安启源Q05 动力参数介绍（纯电版+插混版双体系）
## 一、动力总览
长安启源Q05分为**2026款纯电SUV**、**2025款插电混动SUV**两套动力，均为前置前驱5座紧凑型SUV；纯电搭载400V 3C快充平台，插混搭载智慧新蓝鲸3.0插混系统；全系支持多级动能回收、VTOL对外放电；非营运首任车主三电系统终身质保。

- 电机类型：前置永磁同步油冷电机
- 峰值功率：120kW（163Ps）
- 峰值扭矩：190N·m
- 变速箱：电动车单速固定齿比减速器

- 最高电子限速：160km/h
- 散热：独立油冷散热，持续短途高速无热衰减

## 2. 双续航版本电池、充电、续航对比

- V2L外放电：3.3kW

## 3. 电池热管理与安全
1. 金钟罩磷酸铁锂电池，多层气凝胶隔热，底部加厚钢制防护，IP68防水防尘
2. 全域液冷热泵，-30℃低温自动预热、高温主动散热，电芯温差≤±2℃
3. 碰撞2ms切断高压回路，24小时监测电芯压差、温度；通过针刺、挤压、火烧极限安全测试

## 4. 动能回收系统
- 三档动能回收强度可调，强回收模式支持城市单踏板行驶
- CRBS机电协同制动，90%日常减速依靠电机反向发电储能
- 长下坡自动提升回收功率，稳定车速同步补能

# 第二部分：2025款 启源Q05 插混（智慧新蓝鲸3.0 PHEV）
## 1. 1.5L混动专用发动机（全系通用）
- 型号：全铝合金自吸，多点电喷、冷却EGR
- 排量：1498mL
- 发动机最大功率：72kW（98Ps）/6000rpm
- 发动机峰值扭矩：125N·m /4000rpm
- 燃油标号：92#无铅汽油，主打高效发电、高速直驱

## 2. 前置驱动电机
- 电机类型：前置永磁同步油冷电机
- 峰值功率：158kW（215Ps）
- 峰值扭矩：330N·m
- 传动：E-CVT电控无级混动变速箱，无物理挡位，自动切换纯电/串联增程/并联动力/发动机直驱四大工况
- 系统综合峰值功率：230kW

## 3. 续航、能耗、补能参数
- 电池容量：18.99kWh磷酸铁锂
- CLTC纯电续航：125km，WLTC纯电95km
- CLTC满油满电综合续航：1215km，油箱容积62L
- WLTC官方亏电油耗：5.3L/100km，实测市区亏电4.6–5.2L/100km
- 0-100km/h官方加速：7.3s
- 快充：27kW直流快充，30%-80%约19分钟
- 慢充：6.6kW交流，完整充满约6小时
- V2L外放电：3.3kW

## 4. 蓝鲸3.0插混工作逻辑
1. **低速纯电**：发动机停机，电机单独驱动，市区零油耗、静谧；
2. **串联增程**：低电量发动机发电，电能供给电机驱动车轮；
3. **高速直驱**：高速巡航发动机直接驱动车轮，减少能量转换损耗；
4. **急加速并联**：油电同步输出，提升瞬时动力储备；
5. **滑行/制动回收**：松油门、刹车电机反向发电储存电能。

## 5. 混动动能回收与电控
1. 自适应三档动能回收，经济模式回收力度更强，接近单踏板；运动模式回收减弱，滑行距离更长
2. 液压制动与电机能量回收无缝协同，优先电回收，减少刹车片磨损
3. 长下坡自动拉高回收功率，稳定车速同时补电

# 三、全系通用动力安全防护机制
1. 电机全域油冷独立散热，长时间满载、持续急加速无热衰减
2. 400V高压线束分层绝缘密封，故障瞬间切断高压回路，杜绝漏电风险
3. 24小时实时监测电芯电压、温度、压差，多级电控分级保护：过压/欠压/过流/超温自动限制电机输出功率
4. 插混版发动机多级温控保护，长时间馈电高速、满载爬坡分级限功率输出
