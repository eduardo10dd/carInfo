# 2026款吉利帝豪（第四代）动力技术参数介绍（1.5L自吸 + 1.5T涡轮两套动力）
## 一、整车基础概况
第四代吉利帝豪基于BMA模块化架构打造，定位入门级家用紧凑型三厢轿车，车身尺寸4638×1820×1460mm，轴距2650mm；全系**前置前驱**布局，统一加注92#无铅汽油；分为1.5L高效自吸（走量主力）、1.5T涡轮增压（高性能版）两套动力；底盘统一前麦弗逊独立悬架+后H型扭力梁非独立悬架，主打低油耗、皮实耐用、低维保成本，家用代步定位清晰。

## 第一部分：1.5L高效自吸动力（BHE15-CFN，全系走量）
### 发动机硬件技术
- 型号：BHE15-CFN 1.5L直列四缸中置缸内直喷自然吸气
- 最大功率：93kW（127Ps）/6300rpm
- 峰值扭矩：152N·m /4000–5000rpm
- 核心技术：CVVT连续可变气门正时、低摩擦活塞、电子水泵、可变排量机油泵，最高热效率41%，怠速抖动抑制优秀，长期使用故障率低
- 油箱容积50L，满油综合续航约850km

### 两种变速箱版本
1. **5MT五速手动变速箱**
    WLTC综合油耗5.62L/100km，换挡逻辑清晰，适合追求低成本、手动驾驶用户
2. **模拟8速CVT无级变速箱**
    传动平顺无顿挫，低速蠕行友好，WLTC综合油耗5.82L/100km；提供经济/标准/运动三种驾驶模式

### 性能表现
- 0-100km/h加速：12.3s
- 最高电子限速：185km/h
- 适配场景：城市日常通勤、买菜代步、家用短途，看重省油、低用车成本

## 第二部分：1.5T涡轮增压高性能动力（BHE15-EFZ）
### 发动机硬件技术
- 型号：BHE15-EFZ 1.5L直列四缸涡轮增压
- 最大功率：133kW（181Ps）/5500rpm
- 峰值扭矩：290N·m /1500–4000rpm，1500转即可输出最大扭矩，低速起步、中段超车动力储备充足
- 技术亮点：低惯量涡轮、350MPa高压直喷、冷却EGR，兼顾动力与燃油经济性
- 油箱容积50L

### 传动系统：7速湿式双离合变速箱
湿式结构自带独立油冷散热，长时间拥堵蠕行不易过热；换挡响应更快，支持运动模式拉高转速提升爆发力
### 性能与能耗
- 官方0-100km/h加速：7.9s
- 最高电子限速：200km/h
- WLTC综合油耗5.95L/100km，高速巡航油耗表现优异
- 适配场景：经常满载、高速长途、追求更强加速能力的用户

## 四、底盘、制动与行驶特性
1. **悬架结构**：前麦弗逊独立悬架搭配液压衬套，过滤路面细碎颠簸；后扭力梁非独立悬架，结构简单、维修便宜，底盘调校偏舒适家用
2. **转向系统**：EPS电动助力转向，低速轻盈、高速沉稳，车身短小，小区、窄路停车灵活
3. **制动硬件**：前轮通风盘制动，后轮实心盘制动；标配ESC车身稳定、上坡辅助、牵引力控制、自动驻车，制动踏板线性柔和，新手易上手
4. **NVH优化**：发动机舱隔音棉、底盘隔音涂层，抑制高速风噪与发动机噪音，满足家用静谧需求

## 五、动力安全防护机制
1. 全系发动机水冷循环温控，长时间拥堵、满载爬坡自动限制输出功率，避免高温损伤；涡轮版本配备独立延时冷却回路，熄火后持续散热降低热衰减
2. CVT变速箱内置油冷散热，城市频繁启停不易过热；1.5T湿式双离合变速箱散热油路强化，低速蠕行耐久性更强
3. 多重电控分级保护：爆震、机油压力异常、水温过高自动限制动力输出，保护发动机硬件
4. ESC车身稳定系统联动制动，雨雪湿滑路面抑制车轮打滑、车身推头甩尾，提升家用行驶安全性
