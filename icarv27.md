# iCAR V27 动力参数介绍（超级黄金增程系统）
## 一、动力总览
全系**增程式混动**，1.5T专用增程器仅发电、不直驱车轮，全程电机驱动；分**后置后驱单电机、i-AWD双电机四驱**两套动力；统一34.3kWh宁德时代磷酸铁锂动力电池；加注92#汽油；标配6kW V2L对外放电、全域液冷热泵温控；高配搭载MRC磁流变减振悬架、8+X全地形模式。

## 二、1.5T 增程器（全系通用）
- 型号：SQRH4J15 直列四缸涡轮增压米勒循环增程专用发动机
- 排量：1498mL
- 最大功率：115kW（156Ps）/5500rpm
- 峰值扭矩：220N·m /2000–4000rpm
- 核心热效率：45.79%，1L燃油可发电3.71度；P1发电机效率95.38%
- 工作逻辑：仅发电，不参与驱动车轮，22项NVH优化，怠速运转噪音＜61分贝

## 六、电池全域热管理与安全
1. 宁德时代磷酸铁锂34.3kWh电池包，IP68防尘防水，多层气凝胶隔热防护，通过针刺、挤压、火烧极限安全测试；
2. 全域集成液冷热泵系统，-30℃低温自动预热、高负荷主动散热，电芯温差控制±2℃，降低冬季续航衰减；
3. 24小时实时监测电芯电压、温度、压差，碰撞瞬间切断高压回路，杜绝漏电风险；
4. 三电质保：整车三电8年/15万公里。

## 七、动能回收与电控制动系统
1. 多级可调动能回收强度：节能模式回收力度最强，支持城市单踏板行驶；运动/越野模式回收减弱，滑行距离更长；
2. CRBS机电协同制动，日常减速90%依靠电机反向发电储能，大幅减少刹车片磨损；
3. 长下坡自动提升回收功率，稳定车速同步补电；制动末端舒适刹停调校，抑制点头、晕车。

## 八、增程系统四大工作模式
1. **EV纯电模式**：电量充足时增程器停机，电机单独驱动，市区零油耗、静谧性最优；
2. **串联增程模式**：中低电量增程器稳定高效区间发电，电能供给电机驱动车轮；
3. **并联全力输出**：急加速、爬坡时增程器满功率发电+电池放电，释放系统全部动力；
4. **滑行/制动回收**：松油门、踩刹车电机反向发电，回收能量存入电池。

## 九、动力安全防护机制
1. 前后电机独立全域油冷循环散热，持续激烈驾驶、长途高速、越野工况无热衰减；
2. 1.5T增程器多级智能温控，长时间满载、高温爬坡分级限制输出功率；涡轮水冷熄火延时散热；
3. 400V高压线束分层绝缘密封，高压回路故障毫秒级断电保护；
4. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率；
5. 标配爆胎稳定控制系统，高速爆胎抑制车身跑偏失控。
