# 方程豹豹8 动力参数介绍
基于DMO越野超级混动平台，全系统一2.0T插混双电机四驱，非承载硬派车身，加注92#汽油，标配6kW V2L外放电、三把锁、云辇-P液压底盘；整车6年/15万公里质保，三电首任非营运终身质保。

## 一、全套动力硬件参数
### 1、2.0T纵置涡轮增压发动机（BYD487ZQD）
- 布局：纵置直列四缸，水冷涡轮
- 发动机最大功率：180kW（245Ps）
- 发动机峰值扭矩：380N·m / 1700–4000rpm
- 作用：发电为主，高速可直驱车轮，兼顾越野高负荷稳定输出
- 油箱容积：91L

### 2、前后双永磁同步驱动电机（全系统一）
- 前驱动电机：峰值200kW，峰值扭矩360N·m
- 后桥驱动电机：峰值300kW，峰值扭矩400N·m
- 电机综合总功率：500kW
- 整套系统综合最大功率：550kW（748Ps）
- 系统综合峰值扭矩：760N·m
- 变速箱：DMO专用E‑CVT电子无级混动变速箱，带低速越野21倍放大速比

### 3、性能加速与极速
- 0‑100km/h官方加速：4.8s
- 电子限速最高车速：180km/h

### 4、动力电池（CTC底盘一体化刀片电池）
- 电池类型：磷酸铁锂刀片电池
- 电池容量：36.8kWh
- WLTC纯电续航：100km
- CLTC满油满电综合续航：1200km
- 快充规格：峰值120kW直流快充，30%-80%仅16分钟
- 温控：全域液冷直冷直热，冬季预热、快充均衡控温

### 5、油耗数据
- WLTC综合工况油耗：1.45L/100km
- WLTC馈电综合油耗：8.87L/100km

## 二、四驱与越野动力逻辑
1. **智能电四驱+三把锁**：前/中/后电子差速锁，10ms内完成扭矩锁止，极限交叉轴、泥泞、陡坡脱困；低速挡21倍扭矩放大，攀爬、拖拽动力充沛。
2. **多模式动力分配**：纯电优先、混动均衡、保电、运动、雪地、岩石6种模式；两档动能回收可调。
3. **电驱越野优势**：电机低速瞬时满扭矩输出，起步、爬坡无涡轮迟滞；涉水时高压系统自动绝缘防护，最大涉水深度890mm。
4. **云辇-P联动动力**：液压车身升降时同步调节电机输出扭矩，上坡自动增强动力、下坡加强动能回收。
