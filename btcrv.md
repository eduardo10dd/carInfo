# 2026款东风本田CR-V 动力参数介绍
## 一、动力体系总览
全系三套动力：**240TURBO 1.5T燃油、e:HEV 第五代i-MMD油电混动、e:PHEV插电混动**；提供前置前驱/Real-Time AWD智能适时四驱，全系加注92#无铅汽油，国六b排放；驾驶模式：经济/标准/运动；混动/插混配备多级动能回收。

## 二、240TURBO 1.5T地球梦燃油动力
### 1. 发动机参数（L15BZ）
- 布局：直列四缸涡轮增压、DOHC、双VTC可变气门正时、缸内直喷
- 排量：1498mL
- 最大功率：142kW（193Ps）/6000rpm
- 峰值扭矩：243N·m /1800–5000rpm
### 2. 传动系统
CVT无级变速箱（带起步齿轮），钢带传动，全程无换挡顿挫；配备模拟手动换挡模式。
### 3. 四驱系统 Real-Time AWD
电控多片离合器，铺装路面前驱省油；打滑瞬间向后轮分配动力，无机械锁止，适配雨雪、轻度烂路。
### 4. 性能与能耗
- 0-100km/h加速：9.4s
- WLTC综合油耗：两驱7.25L/100km；四驱7.8L
- 油箱容积53L，满油续航约730km
- 最高电子限速：200km/h

## 四、e:PHEV 插电混动（可外接快充）

同2.0L阿特金森发动机+前后双电机四驱布局
- 前驱动电机135kW/335N·m，后桥新增独立电机
- 系统综合峰值功率243kW（330Ps），综合扭矩550N·m
- 0-100km/h加速：4.9s
### 2. 电池与续航
大容量三元锂电池，CLTC纯电续航82km；支持直流快充+交流慢充
- WLTC综合油耗：1.61L/100km；馈电油耗5.5L
- 满油满电综合续航超1100km
### 3. 外放电
标配2.2kW VTOL对外放电，满足露营小家电使用。

## 五、i-MMD混动核心工作逻辑
1. **低速纯电**：发动机停机，电机单独驱动，市区零油耗、静谧；
2. **串联增程**：中低速发动机发电，电能供给电机驱动车轮；
3. **高速直驱**：高速区间离合器锁止，发动机直接驱动车轮，减少能量转换损耗；
4. **急加速并联**：油电同步输出，瞬时拉高动力储备；
5. **滑行/制动回收**：松油门、踩刹车电机反向发电储能。

## 六、动能回收与电控制动（混动/插混专属）
1. 自适应多级动能回收，经济模式回收力度强，接近单踏板；运动模式回收减弱，滑行距离更长；
2. 液压制动与电机能量回收无缝协同，日常减速优先电回收，减少刹车片磨损；
3. 长下坡自动提升回收功率，稳定车速同时储存电能。

## 七、动力安全防护机制
1. 1.5T地球梦发动机智能温控，长时间拥堵、高速爬坡分级限功率，涡轮水冷延时散热；
2. CVT/E-CVT变速箱独立油冷散热，持续高负荷工况稳定，不易过热保护；
3. 混动/插混高压系统24小时监测电压、温度、电芯压差；碰撞瞬间切断高压回路，杜绝漏电；
4. 多重电控分级保护：过压、欠压、过流、超温自动限制电机输出功率。
